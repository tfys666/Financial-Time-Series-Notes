# 《金融时间序列》复习总结

## 第四章 非平稳时间序列的确定性分析

本章的核心在于处理不再满足“平稳性”前提的时间序列。非平稳序列主要分为两类：**方差非平稳** 和 **期望非平稳**。对于这类序列，我们不能直接使用 ARMA 模型，而必须先提取出其中的“确定性因素”（如长期趋势、季节效应），将其转化为平稳序列后再进行随机性分析。

### 一、 非平稳序列的分类与预处理

**1. 方差非平稳**
*   **定义**：序列的方差不等于常数，即 $D(X_t) = kf(\mu_t) > 0$，方差随着均值 $\mu_t$ 的变化而变化。
*   **处理方法（方差平稳化变换）**：
    我们需要寻找一个转换函数 $h(\cdot)$，使得转换后的序列 $Y_t = h(X_t)$ 的方差是一个常数。
*   **常用转换函数**：
    通过对 $h(X_t)$ 在均值点 $\mu_t$ 处进行一阶 Taylor 展开，要求 $D(Y_t) \approx [h'(\mu_t)]^2 kf(\mu_t) = \text{常数}$，可得转换函数构造公式 $h'(\mu_t) = \frac{1}{\sqrt{f(\mu_t)}}$。
    *   若 $f(\mu_t) = \mu_t$（方差与均值成正比）：取 $h(X_t) = 2\sqrt{X_t}$
    *   若 $f(\mu_t) = \mu_t^2$（标准差与均值成正比）：取 $h(X_t) = \ln(X_t)$
    *   若 $f(\mu_t) = \mu_t^{-2}$：取 $h(X_t) = {X_t}^2$

**2. 期望非平稳**
期望不为常数，存在明显的上升或下降趋势。主要分为两类：
*   **确定性趋势 (Deterministic Trend)**：如 $X_t = \alpha + \beta t + u_t$。可以用原始序列直接减掉拟合出的趋势线来平稳化。
*   **随机性趋势 (Stochastic Trend)**：如随机游走 $X_t = X_{t-1} + \varepsilon_t$。只能通过**差分**来平稳化。
    (注：差分也可以使有确定趋势的序列平稳，但会导致引入移动平均 (MA) 部分，从而增加方差的复杂性。)

### 二、 时间序列分解定理

**1. Wold 分解定理 (1938)**
这是现代时间序列分析理论的灵魂，也是构造 ARMA 模型的理论基础。
*   **定理内容**：对于任何一个离散自协方差（宽）平稳过程 $\{X_t\}$，它都可以分解为两个**不相关的序列之和**：
    $$X_t = V_t + \xi_t$$
    其中：
    *   $V_t$ 是**确定性序列**（如由正弦波表示的周期序列）。
    *   $\xi_t$ 是**随机序列**，且可以表示为白噪声的无限线性组合（即 $MA(\infty)$ 形式）：$\xi_t = \sum_{j=0}^{\infty} \phi_j \varepsilon_{t-j}$。
*   **意义**：平稳变量的动态演化过程可以用线性模型去近似。

**2. Cramer 分解定理 (1961)**
这是 Wold 定理的推广，适用于**非平稳序列**。
*   **定理内容**：任何一个时间序列的波动都可以视为同时受到了**确定性影响**和**随机性影响**的叠加：
    $$X_t = \mu_t + \varepsilon_t = \sum_{j=0}^d \beta_j t^j + \Psi(B)a_t$$
    其中 $\sum \beta_j t^j$ 代表多项式决定的确定性趋势成分，$\Psi(B)a_t$ 代表平稳的零均值随机误差成分。
*   **意义**：指出了非平稳序列产生的机理就在于它所受到的这两方面的影响中，至少有一方面是不稳定的。

### 三、 确定性因素分解模型

**1. 四大影响因素 (T, C, S, I)**
统计学家 W.M. Persons 提出，所有的序列波动都可以归纳为受到以下四大确定性因素的综合影响：
1.  **长期趋势 (Trend, T)**：序列呈现明显的长期递增或递减趋势。
2.  **循环波动 (Circle, C)**：从低到高再到低的周而复始的波动（如经济周期，周期长度常不固定）。
3.  **季节性变化 (Season, S)**：与季节相关的稳定周期性波动（如节假日、季度）。
4.  **随机波动 (Immediate, I)**：其他不能用确定性因素解释的噪音。
*   **改进模型 (X11 思想)**：当观察期不够长（没有包含几个周期），无法准确区分循环 $C$ 和趋势 $T$ 时，常将 $C$ 替换为特殊交易日因素 (Day, D)。

**2. 常用的因素分解模型**
*   **加法模型**：$X_t = T_t + C_t + S_t (D_t) + I_t$ （季节振幅不随趋势的增长而改变，两条趋势包络线平行）。
*   **乘法模型**：$X_t = T_t \times C_t \times S_t (D_t) \times I_t$ （季节振幅随着趋势的递增而按比例放大，呈现喇叭形扩散）。
*   **伪加法模型**：$X_t = T_t \times (S_t + D_t + I_t - 1) $。
*   **对数加法模型**：$\ln X_t = \ln T_t + \ln S_t + \ln D_t + \ln I_t$（实质上是乘法模型的线性化）。

**3. 趋势效应 (T) 的提取**
最常用的是**简单中心移动平均法**。
*   **一般移动平均**：$M(x_t) = \sum_{i=-k}^k \theta_i x_{t-i}, \quad \forall t > k$
*   **简单中心移动平均**：
    * 对一般移动平均函数增加三个强约束条件：**时期对称**、**系数相等**、**系数和为 1**。
    * 例如 5 期中心移动平均：$$M_5(x_t) = \frac{x_{t-2} + x_{t-1} + x_t + x_{t+1} + x_{t+2}}{5}$$
*   **复合移动平均**：
    * 如果移动平均的期数为偶数（如季度数据 4 期，月度数据 12 期），那么通常需要进行**两次偶数期移动平均**才能实现时期对称，记作 $M_{P \times Q}(x_t)$。
    * 例如季度数据的 $2 \times 4$ 复合移动平均：先做 4 期平均，得到的结果由于落在两个季度中间，再做一次 2 期平均即可对齐回原季度，记作 $M_{2 \times 4}(x_t)$。
*   **优良属性**：
    1. 能够有效提取低阶趋势（一阶线性或二阶抛物线趋势）。
    2. 能够实现拟合方差最小。
    3. **能有效消除季节效应**（进行一个完整周期长度的简单移动平均，可以把正负相抵的季节效应彻底滤掉）。
    * 注：移动平均的期数越多，拟合趋势越平滑；期数越少，对近期变化越敏感。

**4. 季节效应 (S) 的提取**
通过移动平均提取出趋势 $T_t$ 后，无论采用何种模型，季节效应提取的核心思想都是分离出消除趋势后的残余序列，并在各季节内平均以消除随机波动的影响。
*   **加法季节效应的提取步骤：**
    1. **从原序列中消除趋势效应**：$y_t = x_t - T_t$ （此时 $y_t$ 中只剩季节 $S_t$ 和随机波动 $I_t$）
    2. **计算序列总均值**：$\bar{y} = \frac{1}{km} \sum_{i=1}^m \sum_{j=1}^k y_{ij}$ （其中 $m$ 是年数，$k$ 是一年内的季节数）
    3. **计算同季节（如某季度/某月）的平均值**：$\bar{y}_j = \frac{1}{m} \sum_{i=1}^m y_{ij}, \quad j=1, 2, \dots, k$
    4. **求季节指数**：同季度均值减去总均值，得到该季节单纯由周期性引起的净增加/减少量：$S_j = \bar{y}_j - \bar{y}$
*   **乘法季节效应的提取步骤：**
    1. **从原序列中消除趋势效应**：$y_t = \frac{x_t}{T_t}$ （此时 $y_t$ 中只剩季节 $S_t \times$ 随机波动 $I_t$）
    2. **计算序列总均值**：$\bar{y} = \frac{1}{km} \sum_{i=1}^m \sum_{j=1}^k y_{ij}$
    3. **计算同季节的平均值**：$\bar{y}_j = \frac{1}{m} \sum_{i=1}^m y_{ij}, \quad j=1, 2, \dots, k$
    4. **求季节指数**：同季度均值除以总均值，得到该季节的相对倍数：$S_j = \frac{\bar{y}_j}{\bar{y}}$
    * 解释：如果某个季度的 $S_j = 1.04$，说明由于季节原因，该季度的平均销售额通常是总均值的 $104\%$，为旺季；若 $S_j < 1$，为淡季。
*   **随机效应的提取**：
    剔除了趋势和季节后，剩下的残余序列即为随机效应：
    加法模型 $I_t = x_t - T_t - S_t$；乘法模型 $I_t = \frac{x_t}{T_t \times S_t}$。

**5. X11 季节调节模型**
*   **研发背景**：基于计算机自动进行的时间序列因素分解方法，广泛用于全球统计机构进行国情监控，排除季节性波动掩盖真实趋势的问题（美国人口普查局研发）。自 X11 之后，后续的 X12 和 X13 模型分别引入了干预分析（包含特殊节假日、闰年、交易日等干预变量）和 seats 季节调整方法。但 X11 模型仍是最重要的理论基础。
*   **X11 模型的核心技术**：通过**三种移动平均方法**进行**三阶段的因素分解**。
    *   **第一次：简单移动平均方法**。
        *   目的：粗略提取趋势和季节效应。
        *   缺陷：对二次以上的曲线趋势信息提取不充分（拟合高阶多项式不精确）；并且会造成首尾数据信息丢失。
    *   **第二次：Henderson 加权移动平均**。
        *   目的：克服简单移动平均无法拟合高阶曲线的缺点。
        *   原理：在约束条件 $\sum_{i=-k}^k \theta_i = 1$ 和 $\sum_{i=-k}^k i\theta_i = 0$ 的前提下，使得移动平均系数的**三阶差分的平方和达到最小**：
            $$S^2 = \sum_{i=-k}^k (\nabla^3 \theta_i)^2 \rightarrow \min$$
        *   意义：这等价于把某个三次多项式作为光滑度的一个指标，**力求修匀值接近一条三次曲线**，极大地提高了对复杂非线性趋势的拟合精度。
    *   **第三次：Musgrave 非对称移动平均**。
        *   目的：克服前两种移动平均作为中心移动平均导致的**两端（首部和尾部各 $k$ 期）数据信息严重丢失**的缺点。
        *   原理：构造一组**非中心移动平均系数**，不仅要求其系数和为 1，且要求其拟合值能**无限接近中心移动平均的拟合值**，即对中心移动平均现有估计值做出的修正最小：
            $$E(\hat{y}_t^m - \hat{y}_t)^2 \rightarrow \min$$
            （其中 $\hat{y}_t^m$ 为 Musgrave 估计值，$\hat{y}_t$ 为原估计值）。
        *   意义：专门用于对最后 $k$ 期（以及最前 $k$ 期）数据进行补充平滑拟合，挽救信息损失。
*   **X11 模型的三阶段迭代过程（以月度数据乘法模型为例）**：
    *   **第一阶段迭代（初步提取季节和剔除异常值）**：
        1. 使用 $2 \times 12$ 复合中心移动平均估计初始趋势：$T_t^{(1)} = M_{2 \times 12}(x_t)$。
        2. 剔除趋势，提取包含季节和随机效应的序列：$S_t^{(1)} I_t^{(1)} = \frac{x_t}{T_t^{(1)}}$。
        3. 对同月数据进行 $3 \times 3$ 移动平均，分离初步季节因子：$\bar{S}_t^{(1)} = M_{3 \times 3}(S_t^{(1)} I_t^{(1)})$。
        4. 归一化初步季节因子：$S_t^{(1)} = \frac{\bar{S}_t^{(1)}}{M_{2 \times 12}(\bar{S}_t^{(1)})}$。
        5. 提取初步随机因子：$I_t^{(1)} = \frac{S_t^{(1)} I_t^{(1)}}{S_t^{(1)}}$。
        6. 根据控制限（如 $\pm 1.5\sigma, \pm 2.5\sigma$）剔除 $I_t^{(1)}$ 中的异常值，得到修正后的随机因子 $\bar{I}_t^{(1)}$，继而重构第一阶段的修正序列：$x_t^{(1)} = T_t^{(1)} S_t^{(1)} \bar{I}_t^{(1)}$。
    *   **第二阶段迭代（引入 Henderson 移动平均，提纯季节效应）**：
        1. 采用 Henderson 加权移动平均（记为 $H_{m_1}$），重新提取趋势因子：$T_t^{(2)} = H_{m_1}(x_t^{(1)})$。
        2. 重新提取季节指数和随机成分：$S_t^{(2)} I_t^{(2)} = \frac{x_t^{(1)}}{T_t^{(2)}}$。
        3. 对同月数据采用更长的 $3 \times 5$ 移动平均，重新分离季节因子：$\bar{S}_t^{(2)} = M_{3 \times 5}(S_t^{(2)} I_t^{(2)})$。
        4. 归一化第二阶段季节因子：$S_t^{(2)} = \frac{\bar{S}_t^{(2)}}{M_{2 \times 12}(\bar{S}_t^{(2)})}$。
        5. 提取第二阶段随机因子：$I_t^{(2)} = \frac{S_t^{(2)} I_t^{(2)}}{S_t^{(2)}}$。
        6. 再次剔除异常值，得到更准确的修匀随机项 $\bar{I}_t^{(2)}$，由此重构第二阶段的修正序列：$x_t^{(2)} = T_t^{(2)} S_t^{(2)} \bar{I}_t^{(2)}$。
    *   **第三阶段迭代（组合 Musgrave，得到最终分解）**：
        1. 再次采用 Henderson 加权移动平均（$H_{m_2}$），得到最终平滑的高阶趋势估计：$T_t^{(3)} = H_{m_2}(x_t^{(2)})$。
        2. 去除最终趋势后的序列：$S_t^{(3)} I_t^{(3)} = \frac{x_t^{(2)}}{T_t^{(3)}}$。
        3. 继续用同月的 $3 \times 5$ 移动平均分离季节成分：$\bar{S}_t^{(3)} = M_{3 \times 5}(S_t^{(3)} I_t^{(3)})$。
        4. 归一化得到最终稳定的季节指数：$S_t = \frac{\bar{S}_t^{(3)}}{M_{2 \times 12}(\bar{S}_t^{(3)})}$。
        5. 分离出最终纯净的随机扰动序列：$I_t = \frac{S_t^{(3)} I_t^{(3)}}{S_t}$。

### 四、 趋势分析
有些时间序列具有非常显著的趋势，我们分析的目的就是要找到序列中的这种趋势，并利用这种趋势对序列的发展作出合理的预测。常用方法包括**趋势拟合法**和**平滑法**。

**1. 趋势拟合法**

把时间 $t$ 作为自变量，相应的序列观察值 $X_t$ 作为因变量，建立序列值随时间变化的回归模型。分为线性拟合与非线性拟合。

*   **线性拟合**：
    *   **适用场合**：长期趋势呈现出线性特征。
    *   **模型结构**：
        $$X_t = a + bt + I_t$$
        其中 $E(I_t) = 0, \ Var(I_t) = \sigma^2$。
    *   **参数估计**：使用线性最小二乘估计。

*   **非线性拟合**：
    *   **适用场合**：长期趋势呈现出非线性特征。
    *   **参数估计指导思想**：能转换成线性模型的都转换成线性模型，用线性最小二乘法进行参数估计；实在不能转换成线性的，就用迭代法进行参数估计。
    *   **常用非线性模型及其转换**：
        | 模型 | 变换方式 | 变换后的线性模型 | 估计方法 |
        | :--- | :--- | :--- | :--- |
        | **二次抛物线** <br> $T_t = a + bt + ct^2$ | $t_1 = t$<br>$t_2 = t^2$ | $T_t = a + bt_1 + ct_2$ | 线性最小二乘估计 |
        | **指数曲线** <br> $T_t = ab^t$ | $T_t' = \ln T_t$<br>$a' = \ln a$<br>$b' = \ln b$ | $T_t' = a' + b't$ | 线性最小二乘估计 |
        | **修正指数曲线** <br> $T_t = a + bc^t$ | 无 | 无法线性化 | 迭代法 |
        | **Gompertz 曲线** <br> $T_t = ab^{c^t}$ | 无 | 无法线性化 | 迭代法 |
        | **Logistic 曲线** <br> $T_t = \frac{1}{a + bc^t}$ | 无 | 无法线性化 | 迭代法 |

**2. 平滑法**

平滑法是利用修匀技术，削弱短期随机波动对序列的影响，使序列平滑化，从而显示出长期趋势变化的规律。常用方法包括**移动平均法**和**指数平滑法**（指数平滑法见下一节）。

*   **移动平均法**
    *   **基本思想**：假定在一个比较短的时间间隔里，序列值之间的差异主要是由随机波动造成的。根据这种假定，用一定时间间隔内的**平均值**作为某一期的估计值。
    *   **分类与计算**：
        *   **$n$ 期中心移动平均**：
            * 当 $n$ 为奇数时：$\tilde{x}_t = \frac{x_{t-\frac{n-1}{2}} + \cdots + x_t + \cdots + x_{t+\frac{n-1}{2}}}{n}$
            * 当 $n$ 为偶数时：$\tilde{x}_t = \frac{\frac{1}{2}x_{t-\frac{n}{2}} + x_{t-\frac{n}{2}+1} + \cdots + x_{t+\frac{n}{2}-1} + \frac{1}{2}x_{t+\frac{n}{2}}}{n}$
        *   **$n$ 期（简单）移动平均**：
            $\tilde{x}_t = \frac{x_t + x_{t-1} + \cdots + x_{t-n+1}}{n}$
    *   **移动平均期数 $n$ 确定的原则**：
        1. **事件的发展有无周期性**：以周期长度作为移动平均的间隔长度 $n$，以消除周期效应的影响。
        2. **对趋势平滑的要求**：移动平均的期数 $n$ 越多，拟合趋势越平滑。
        3. **对趋势反映近期变化敏感程度的要求**：移动平均的期数 $n$ 越少，拟合趋势越敏感。
    *   **移动平均预测**：
        向后 $l$ 期的预测值 $\hat{x}_{T+l}$ 等于最后 $n$ 期数据的简单平均：
        $$\hat{x}_{T+l} = \frac{x_{T+l-1}' + x_{T+l-2}' + \cdots + x_{T+l-n}'}{n}$$
        其中 $x_{T+l-i}' = \begin{cases} \hat{x}_{T+l-i}, & l > i \\ x_{T+l-i}, & l \le i \end{cases}$

### 五、 指数平滑预测模型

平滑法是一种不考虑模型生成机制，纯粹利用修匀技术削弱短期随机波动，从而预测未来的经验方法。在实际生活中，大多数随机事件都是近期的结果对现在的影响会大些，远期的结果对现在的影响会小些。考虑到时间间隔对事件发展的影响，各期权重随时间间隔的增大而呈指数衰减，这就是指数平滑法的基本思想。

**1. 简单指数平滑**
*   **适用场景**：既**无长期趋势**，也**无季节效应**的序列（即序列在均值附近随机波动）。对该序列进行预测的主要目的是消除随机波动的影响，得到序列稳定的均值。
*   **核心思想**：时间越近的观察值对未来的影响越大，各期权重 $\alpha(1-\alpha)^j$ 随时间间隔 $j$ 呈指数衰减，且权重之和为 1。即 $E(\hat{x}_{t+1}) = \sum_{k=0}^\infty \alpha(1-\alpha)^k \mu = \mu$，保证了预测值的**无偏性**。
*   **递推预测公式**：
    $$\hat{x}_{t+1} = \alpha x_t + \alpha(1-\alpha) x_{t-1} + \alpha(1-\alpha)^2 x_{t-2} + \cdots = \alpha x_t + (1-\alpha) \hat{x}_t$$
    *   $\alpha$ 为平滑系数 $(0 < \alpha < 1)$。变化缓慢的序列取小值 $\alpha$，变化迅速的序列取大值 $\alpha$（一般经验在 0.05 ~ 0.3 之间，修匀效果比较好）。
    *   **初始值的确定**：通常取 $\hat{x}_1 = x_1$ 或 $\hat{x}_1 = \tilde{x}_0$（前几期的平均值）。
*   **预测性质**：任意向后 $l$ 期的预测值**均为常数**：$\hat{x}_{T+l} = \hat{x}_{T+1}, \quad \forall l \ge 1$。

**2. Holt 两参数指数平滑**
*   **适用场景**：含有**线性递增/递减趋势**，但无季节效应的序列。即 $x_t = a_0 + bt + \varepsilon_t$。
*   **等价表达**：$$x_t = a_0 + b(t-1) + b + \varepsilon_t$$
$$= (x_{t-1} - \varepsilon_{t-1}) + (b + \varepsilon_t)$$
$$= a(t-1) + b(t)$$
*   **核心思想**：具有线性趋势的序列其水平（截距）和趋势（斜率）是随时间变化的。不仅对截距（水平）进行平滑，还引入参数对斜率（趋势）进行平滑。$a(t-1)$ 代表序列在 $t-1$ 时刻截距的无偏估计值，$b(t)$ 代表序列在 $t$ 时刻斜率的无偏估计值。
*   **平滑与预测公式**：
    1. 平滑截距 $a_t$：$\hat{a}(t) = \alpha x_t + (1-\alpha)[\hat{a}(t-1) + \hat{b}(t-1)]$
    2. 平滑斜率 $b_t$：$\hat{b}(t) = \beta[\hat{a}(t) - \hat{a}(t-1)] + (1-\beta)\hat{b}(t-1)$
    3. 向前 $k$ 期的预测值：$\hat{x}_{t+k} = \hat{a}(t) + \hat{b}(t) \cdot k$
*   **初始值的确定**：
    *   平滑序列初始值：$\hat{a}(1) = x_1$。
    *   趋势序列初始值：$\hat{b}(1) = \frac{x_{n+1} - x_1}{n}$ （利用前 $n+1$ 个数据计算平均斜率）。

**3. Holt-Winters 三参数指数平滑**
*   **适用场景**：同时具有**长期趋势**和**季节效应**的序列。
*   **核心思想**：在截距 $a(t)$ 和斜率 $b(t)$ 的基础上，引入第三个参数 $c(t)$ 对**季节指数 $S(t)$** 进行平滑迭代（$c(t)$ 是对季节指数的无偏估计）。
*   **加法模型的预测公式（季节效应恒定平行）**：
    *   模型等价表达：$x_t = a(t-1) + b(t-1) + c(t-m) + \varepsilon_t$
    1. 平滑截距：$\hat{a}(t) = \alpha (x_t - \hat{c}(t-m)) + (1-\alpha)[\hat{a}(t-1) + \hat{b}(t-1)]$
    2. 平滑斜率：$\hat{b}(t) = \beta[\hat{a}(t) - \hat{a}(t-1)] + (1-\beta)\hat{b}(t-1)$
    3. 平滑季节指数：$\hat{c}(t) = \gamma (x_t - \hat{a}(t)) + (1-\gamma)\hat{c}(t-m)$
    4. 向前 $k$ 期的预测值：$\hat{x}_t(k) = \hat{a}(t) + \hat{b}(t) \cdot k + \hat{c}(t+k)$
*   **乘法模型的预测公式（季节效应随趋势扩大）**：
    *   模型等价表达：$x_t = [a(t-1) + b(t-1)]c(t-m) + \varepsilon_t$
    1. 平滑截距：$\hat{a}(t) = \alpha \frac{x_t}{\hat{c}(t-m)} + (1-\alpha)[\hat{a}(t-1) + \hat{b}(t-1)]$
    2. 平滑斜率：$\hat{b}(t) = \beta[\hat{a}(t) - \hat{a}(t-1)] + (1-\beta)\hat{b}(t-1)$
    3. 平滑季节指数：$\hat{c}(t) = \gamma \frac{x_t}{\hat{a}(t)} + (1-\gamma)\hat{c}(t-m)$
    4. 向前 $k$ 期的预测值：$\hat{x}_t(k) = [\hat{a}(t) + \hat{b}(t) \cdot k] \times \hat{c}(t+k)$
