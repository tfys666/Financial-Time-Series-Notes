本项目是针对《金融时间序列》课程的期末复习总结与核心知识点梳理。内容以 Ruey S. Tsay 的经典著作《金融时间序列分析（第3版）》核心参考，致力于将繁杂的理论推导、平稳性检验、ARMA建模、波动率模型（ARCH/GARCH）以及非平稳序列和协整理论进行系统化的提炼。

旨在帮助学习金融数学、统计学及量化金融的同学快速把握课程脉络，理清公式背后的统计学逻辑，并为期末考试或实际的数据分析工作提供一份结构清晰、重点突出的参考笔记。
