# 《金融时间序列》复习总结

## 第二章 时间序列的预处理

### 核心知识点与公式总结

**1. 平稳时间序列的定义**
*   要判断时间序列的平稳性，首先要考察其概率分布和特征统计量。设时间序列为 $\{X_t\}$：
    *   **均值**：$\mu_t = E(X_t) = \int_{-\infty}^{+\infty} x dF_t(x)$
    *   **方差**：$DX_t = E[(X_t - \mu_t)^2] = \int_{-\infty}^{+\infty} (x - \mu_t)^2 dF_t(x)$
    *   **自协方差**：$\gamma(t, s) = E[(X_t - \mu_t)(X_s - \mu_s)]$
*   **严平稳 (Strictly Stationary)**：一种条件比较苛刻的平稳性定义，认为只有当过程**所有的统计性质（有限维联合概率分布）都不会随时间的推移而发生变化**时，该随机过程才能被认为平稳。即 $F_{t_1, \cdots, t_m}(x_1, \cdots, x_m) = F_{t_1+\tau, \cdots, t_m+\tau}(x_1, \cdots, x_m)$。
*   **宽平稳 (Weakly Stationary)**：也称弱平稳。使用随机过程的特征统计量来定义，只要保证**低阶矩（一阶和二阶矩）平稳**，就能保证序列的主要性质近似稳定。必须同时满足以下三个条件：
    1.  均值为常数：$E(X_t) = \mu$ （$\mu$ 为常数）
    2.  方差为常数：$Var(X_t) = \gamma(0)$ 且有限。
    3.  自协方差函数只依赖于时间的平移长度（滞后期 $k$），而与起止点无关：$Cov(X_t, X_{t-k}) = \gamma(k)$
*   **严平稳与宽平稳的关系**：
    *   宽平稳对时间的不变性表现在一、二阶矩上，严平稳表现在概率分布上。
    *   **严平稳 $\nRightarrow$ 宽平稳**：因为宽平稳要求期望和协方差存在。例如服从柯西分布的严平稳序列，其一、二阶矩不存在，因此不是宽平稳。
    *   **宽平稳 $\nRightarrow$ 严平稳**。显然。
    *   **严平稳 + 二阶矩存在 $\Rightarrow$ 宽平稳**。
    *   **对于正态过程，严平稳 $\Leftrightarrow$ 宽平稳**。实际应用中研究最多的是宽平稳。

**2. 平稳时间序列的统计性质**
*   **自相关系数 $\rho_k = \frac{\gamma_k}{\gamma_0}$ 的性质**：
    1.  **规范性**：$\rho_0 = 1, |\rho_k| \le 1$ （所有 $\rho_k$ 都在 -1 到 1 之间）
    2.  **对称性**：$\rho_k = \rho_{-k}$
    3.  **非负定性**：自相关矩阵正定。
    4.  **非唯一性**：一个平稳序列唯一决定它的 ACF，但一个 ACF 未必唯一对应一个平稳序列。
*   **平稳性的重大意义**：极大地减少了待估随机变量的个数。序列均值等于常数，使得原本只能依靠唯一一个样本估计的均值/方差/自相关系数，现在**拥有了大量的样本观察值**，极大地提高了对特征统计量的估计精度。

**3. 随机游走模型 (Random Walk) 与伪回归（为什么要做平稳性检验）**
*   **经典回归模型（OLS）假定**：暗含了数据是平稳的，要求大样本下统计推断满足“一致性”。
*   **随机游走**：模型为 $X_t = X_{t-1} + \mu_t$ （其中 $\mu_t$ 是白噪声）。由于 $Var(X_t) = t \sigma^2$，其**方差随时间 $t$ 增大而增大，因此是一非平稳序列**。
*   **伪回归 (Spurious Regression)**：如果时间序列是有趋势的（非平稳），采用 OLS 估计的 t 检验和 F 检验就是无效的。两个毫无关系但具有相同趋势的时间序列，在回归时也可能得到很高的显著性和复相关系数 $R^2$。
*   **解决办法**：出现伪回归时，要么加入趋势变量，要么把非平稳的序列**平稳化（通常通过差分）**。

**4. 时间序列平稳性的检验方法**

*   **① 时序图检验（直观判断）**
    *   **平稳序列**：时序图应表现出该序列始终在**一个常数值附近随机波动**，波动的范围有界、无明显趋势及周期特征。
    *   **非平稳序列**：往往表现出在不同时间段有不同的均值（如持续上升或下降趋势）。

*   **② 样本自相关函数 (ACF) 图检验**
    *   **公式**：$\hat{\rho}_k = \frac{\sum_{t=1}^{n-k}(X_t - \bar{X})(X_{t+k} - \bar{X})}{\sum_{t=1}^n (X_t - \bar{X})^2}$
    *   **平稳序列**：ACF 随延迟期数 $k$ 的增加会**迅速衰减向零**。
    *   **非平稳序列**：ACF 随延迟期数 $k$ 的增加**缓慢下降**。

*   **③ 平稳性的单位根检验 (Unit Root Test)**

    这是最准确和重要的统计检验方法，通过检验特征根是否在单位圆内（或等于 1）来判断平稳性。

    **方法 A：基础 DF 检验 (Dickey-Fuller Test)**
    *   **推导**：针对一阶自回归 AR(1)：$X_t = \rho X_{t-1} + \mu_t$。如果 $\rho=1$，则为随机游走，非平稳。对式子做一阶差分，得到等价检验形式：$\Delta X_t = \delta X_{t-1} + \mu_t$ （其中 $\delta = \rho - 1$）。
    *   **假设检验**：原假设 $H_0: \delta = 0$（存在单位根，序列**非平稳**）；备择假设 $H_1: \delta < 0$（序列**平稳**）。
    *   **检验统计量**：由于 $t$ 统计量在原假设下有向下偏倚性，它服从特殊的 **DF 分布**。计算的统计量称为 $\tau$ 统计量。
    *   **判别**：DF 检验是**左尾单侧检验**。如果计算的 $\tau$ 统计量 **$<$ 临界值**（如 $-3.5 < -2.89$），则拒绝原假设，序列**平稳**；如果 $\tau$ 统计量 $\ge$ 临界值（或为正数），则接受原假设，序列**非平稳**。

    **方法 B：ADF 检验 (Augmented Dickey-Fuller Test)**
    *   **提出的背景**：DF 检验只在序列为 AR(1) 即扰动项为独立同分布的白噪声时有效。如果序列存在**高阶滞后相关**，DF 检验将无效。ADF 通过在方程右侧加入**高阶滞后差分项 $\sum_{i=1}^m \beta_i \Delta X_{t-i}$** 以消除残差的序列相关性。
    *   **ADF 检验的三个模型**（实际检验顺序：模型 3 $\rightarrow$ 模型 2 $\rightarrow$ 模型 1）：
        *   模型 3（含常数项和时间趋势 $t$）：$\Delta X_t = \alpha + \beta t + \delta X_{t-1} + \sum_{i=1}^m \beta_i \Delta X_{t-i} + \varepsilon_t$
        *   模型 2（含常数项 $\alpha$，无趋势）：$\Delta X_t = \alpha + \delta X_{t-1} + \sum_{i=1}^m \beta_i \Delta X_{t-i} + \varepsilon_t$
        *   模型 1（无常数项，无趋势）：$\Delta X_t = \delta X_{t-1} + \sum_{i=1}^m \beta_i \Delta X_{t-i} + \varepsilon_t$
    *   **检验标准**：同样检验 $H_0: \delta=0$。**只要其中有一个模型的检验结果拒绝了零假设（计算值 $<$ 对应临界值），就可以认为时间序列是平稳的**；当三个模型的检验结果都不能拒绝零假设时，认为时间序列是非平稳的。
    *   **滞后阶数 $m$ 的选择**：可采用渐进 t 检验（从大向小减 $m$ 直到系数显著）或 AIC/SC 信息准则来确定。

    **方法 C：PP 检验 (Phillips-Perron Test)**
    *   ADF 检验主要适用于方差齐性场合，它对于**存在异方差**序列的平稳性检验效果不佳。PP 检验通过非参数修正，**适用于异方差场合**的平稳性检验，且服从相应的 ADF 检验极限分布。
    *   只有干扰项同方差且没有序列相关才可以用ADF，当存在异方差或序列相关用PP比较准确。

**5. 白噪声序列（纯随机序列）的定义与性质**
*   **定义**：时间序列 $\{\varepsilon_t\}$ 称为白噪声序列，如果它是一个具有**有限均值**和**有限方差**的**独立同分布**随机变量序列。若服从正态分布，则称高斯白噪声。
    数学表示：$E(\varepsilon_t) = \mu$；$Var(\varepsilon_t) = \sigma^2$；$Cov(\varepsilon_t, \varepsilon_s) = 0 \ (t \neq s)$。
*   **性质**：
    1.  **纯随机性**：各序列值之间没有任何相关关系（$\gamma(k)=0, \forall k \neq 0$），即为“没有记忆”的序列，完全无序波动。**不需要也无法进行时间序列建模**。
    2.  **方差齐性**：根据马尔可夫定理，只有方差齐性假定成立时，最小二乘法估计才是有效的。

**6. 纯随机性检验（基于 Bartlett 定理的假设检验）**
*   **前提**：只有通过**平稳性检验后**才能进行纯随机性检验。
*   **Bartlett 定理**：如果序列是纯随机的，长度为 $n$ 的样本，其非零期的样本自相关系数将近似服从均值为 0，方差为 $1/n$ 的正态分布：$\hat{\rho}_k \sim N(0, \frac{1}{n})$。
*   **假设条件**：
    *   原假设 $H_0: \rho_1 = \rho_2 = \cdots = \rho_m = 0$ （延迟 $m$ 期内序列值互不相关，即该序列是白噪声）。
    *   备择假设 $H_1:$ 至少存在某个 $\rho_k \neq 0$ （序列非纯随机）。
*   **检验统计量**：
    *   **Q 统计量 (Box-Pierce)**：$Q = n \sum_{k=1}^m \hat{\rho}_k^2 \sim \chi^2(m)$
    *   **LB 统计量 (Ljung-Box)**（实际中最常用，小样本下更精确）：$LB = n(n+2) \sum_{k=1}^m \frac{\hat{\rho}_k^2}{n-k} \sim \chi^2(m)$
*   **判别原则**：
    *   **拒绝原假设**：计算的 $LB$ 统计量 $> \chi^2_\alpha(m)$ 分位点，或 $P$ 值 $< \alpha$ (如 $0.05$)。认为该序列**不是白噪声**，有序列相关性，有建模提取信息的价值。
    *   **接受原假设**：计算的 $LB$ 统计量 $< \chi^2_\alpha(m)$ 分位点，或 $P$ 值 $> \alpha$。认为序列在 $\alpha$ 置信水平下是**白噪声（纯随机）**，无需继续建模。
