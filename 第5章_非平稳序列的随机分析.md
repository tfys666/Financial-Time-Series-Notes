# 《金融时间序列》复习总结

## 第5章 非平稳序列的随机分析

### 一、 差分运算
1. **差分运算的实质与理论保证**
    *   **理论基础**：Cramer 分解定理在理论上保证了适当阶数的差分一定可以充分提取确定性信息。
    *   **实质**：使用自回归的方式提取确定性信息。
        *   $p$ 阶差分公式：$\nabla^d X_t = (1-B)^d X_t = \sum_{i=0}^d (-1)^i C_d^i X_{t-i}$。
2. **差分方式的选择逻辑**
    *   **线性趋势**：序列蕴含显著的线性递增/递减趋势，**一阶差分**即可实现趋势平稳。
    *   **曲线趋势**：序列蕴含曲线趋势，通常**低阶（二阶或三阶）差分**就可以提取出曲线趋势的影响。
    *   **固定周期**：对于蕴含着固定周期的序列，进行步长为周期长度（如 12 步）的**季节差分**运算，可以较好地提取周期信息。
    *   **异方差趋势（随时间扩大）**：若方差随均值发散，直接差分仍无法平稳波动，必须**先取对数，再做一阶差分**，方可实现方差齐性和趋势平稳。
3. **过差分问题**
    *   **现象与后果**：差分运算的阶数并不是越多越好。每次差分都会伴随信息的损失，过度差分不仅会引入多余的短期相关性，还会**显著放大序列的方差**。
    *   **公式推导与证明**：
        
        假设原序列模型为：$X_t = \beta_0 + \beta_1 t + a_t$ 
        
        其中白噪声 $E(a_t) = 0, Var(a_t) = \sigma^2, Cov(a_t, a_{t-i}) = 0 \ (\forall i \ge 1)$。
        *   **一阶差分**：提取线性趋势，得到平稳序列。
            
            $\nabla X_t = X_t - X_{t-1} = \beta_1 + a_t - a_{t-1}$
            
            此时方差为：$Var(\nabla X_t) = Var(a_t - a_{t-1}) = 2\sigma^2$。
        *   **二阶差分（过差分）**：序列仍然平稳，但引入了更高阶的噪声叠加。
            
            $\nabla^2 X_t = \nabla X_t - \nabla X_{t-1} = a_t - 2a_{t-1} + a_{t-2}$
            
            此时方差剧增：$Var(\nabla^2 X_t) = Var(a_t - 2a_{t-1} + a_{t-2}) = 6\sigma^2$。
    *   **结论**：因为 $Var(X_t - \beta_0 - \beta_1 t) = \sigma^2 < 2\sigma^2 < 6\sigma^2$，所以**能用确定性模型就用确定性模型**拟合；差分阶数越小越好，坚决避免过差分。

### 二、 ARIMA 模型
1. **模型定义与适用场合**
    *   **适用场合**：用于差分平稳序列的拟合。
    *   **模型结构 ARIMA(p,d,q)**：
        $$\Phi(B) \nabla^d X_t = \Theta(B)\varepsilon_t$$
        其中 $E(\varepsilon_t)=0, Var(\varepsilon_t)=\sigma^2, E(\varepsilon_t \varepsilon_s)=0 \ (\forall s \neq t)$。
    *   **特例模型族**：
        *   $d=0$ 时退化为平稳的：ARIMA(p,0,q) = ARMA(p,q)
        *   $p=0$ 时：ARIMA(0,d,q) = IMA(d,q)
        *   $q=0$ 时：ARIMA(p,d,0) = ARI(p,d)
        *   $d=1, p=q=0$ 时退化为**随机游走模型 (Random Walk)**：$X_t = X_{t-1} + \varepsilon_t$。（常用于描述投机价格走势，是有效市场理论的核心）。
2. **模型的性质**
    *   **非平稳性**：ARIMA(p,d,q) 共有 $p+d$ 个特征根，其中 $p$ 个在单位圆内，**$d$ 个刚好在单位圆上**。所以当 $d \neq 0$ 时，模型一定非平稳。
    *   **方差非齐性**：原序列方差非齐性（随时间发散）。例如随机游走模型 $Var(X_t) = t\sigma^2$，但一阶差分后的序列方差齐性（$Var(\nabla X_t) = \sigma^2$）。
3. **建模与漂移项设定 (Drift)**
    *   **建模步骤**：原序列 $\to$ 平稳性检验 (不平稳) $\to$ 差分运算 $\to$ 白噪声检验 (非白噪声) $\to$ 拟合 ARMA 模型 $\to$ 分析结束。
    *   **漂移项的核心影响**：
        *   在有差分运算的场合，漂移项指的是**差分后序列的均值**：$\mu = \mathbb{E}\big[(1 - B)^d x_t\big]$。
        *   **缺省漂移项**：不影响序列间相关性的提取，但**会严重影响拟合模型的预测均值和置信区间**。无漂移项预测值往往趋于常数（扁平）；带有漂移项的预测模型则会顺应历史演变呈现合理的线性递增/递减，长期来看预测效果显著更优。
4. **ARIMA 模型的预测**
    *   **原模型多项式展开 (Green 函数表达)**：将模型写成仅依赖随机扰动项 $\varepsilon_t$ 的线性函数形式：$X_t = \sum_{j=0}^\infty \psi_j \varepsilon_{t-j}$。
    *   **Green 函数 $\psi_j$ 待定系数递推公式**：
        *   $\psi_1 = \phi_1 - \theta_1$
        *   $\psi_2 = \psi_1 \phi_1 + \phi_2 - \theta_2$
        *   ...
        *   $\psi_j = \psi_{j-1}\phi_1 + \dots + \phi_{p+d}\psi_{j-p-d} - \theta_j$
    *   **求解 $l$ 步预测及置信区间**：
        1. 展开原模型多项式项，写出 $X_{t+l}$ 关于历史观察值和历史误差项的表达式。
        2. 求条件期望获得 $\hat{X}_t(l)$，并将未来的误差项 $E(\varepsilon_{t+k})=0$ 代入消去。
        3. 利用 Green 函数系数求 $l$ 步预测误差的方差：$Var[e_t(l)] = \sigma^2 \sum_{j=0}^{l-1} \psi_j^2$。（其中 $\psi_0=1$）。
        4. 构造 95% 置信区间：$[\hat{X}_t(l) - 1.96\sqrt{Var[e_t(l)]}, \hat{X}_t(l) + 1.96\sqrt{Var[e_t(l)]}]$。
5. **疏系数模型**
    *   **概念与适用场景**：常规的 ARIMA(p,d,q) 模型默认包含 $1 \sim p$ 阶的所有自回归项和 $1 \sim q$ 阶的所有移动平均项。但在实际应用中，差分平稳后序列的 ACF 或 PACF 往往呈现出**不连续的显著特征**（即中间某些阶数的系数落在两倍标准差以内，统计不显著）。为了避免引入冗余参数导致模型精度下降，应当把这些不显著的滞后项剔除，建立部分系数缺省的“疏系数模型”。
    *   **记号表达与分类**：采用双层括号 `( )` 来明确罗列保留下来的**非零阶数**。
        *   **仅自相关疏系数**：记为 $ARIMA((m_1, m_2, \dots), d, q)$。例如 $ARIMA((1,4), 1, 0)$，表示序列经 1 阶差分后，只包含滞后 1 阶和 4 阶的 AR 项（即有 $\phi_1, \phi_4$，缺省 $\phi_2, \phi_3$）。
        *   **仅移动平均疏系数**：记为 $ARIMA(p, d, (n_1, n_2, \dots))$。表示只保留特定离散的 MA 项。
        *   **双向疏系数**：自相关和移动平均都有缺省，记为 $ARIMA((m_1, \dots), d, (n_1, \dots))$。
    *   **定阶判别套路**：直接观察差分后序列的 ACF 和 PACF 图，精准挑出**显著跳出两倍标准差红线**的离散阶数进行组合拟合。
6. **季节模型 (Seasonal ARIMA)**
    *   **季节加法模型**：季节效应和其他效应（趋势、随机波动）是简单加法关系。简单经过 $d$ 阶趋势差分、$D$ 阶季节（周期为 $S$）差分后即可转化为平稳。
        *   结构方程：$\Phi(B) \nabla^d \nabla_S^D X_t = \Theta(B)\varepsilon_t$。（其中 $S$ 为周期长度，如季度数据 $S=4$，月度数据 $S=12$）
    *   **季节乘法模型**：效应之间存在复杂互动关联，加法模型不足以充分提取。
        *   **构造逻辑**：
            1.  短期相关性：用低阶 $ARMA(p,q)$ 模型提取，对应多项式为 $\Phi(B)$ 和 $\Theta(B)$。
            2.  季节相关性：用以周期步长 $S$ 为单位的 $ARMA(P,Q)$ 模型提取，对应多项式为 $\Phi_S(B)$ 和 $\Theta_S(B)$。
        *   结构方程：$\Phi(B) \Phi_S(B^S) \nabla^d \nabla_S^D X_t = \Theta(B) \Theta_S(B^S) \varepsilon_t$。
        （通常简记为 $ARIMA(p,d,q) \times (P,D,Q)_S$ 模型）
        *   **判别套路**：差分后，看 ACF/PACF 图。如果延迟 1、2 阶等连续前几阶超出 2 倍标准差，说明存在短期相关；如果以 12、24 为单位的周期滞后阶数显著超出 2 倍标准差，说明存在长期季节效应。最终将两者对应的 ARMA 多项式相乘拟合。

### 三、 Auto-Regressive 模型
1. **构造思想**
    *   首先通过**确定性因素分解方法**提取序列中主要的确定性信息：$X_t = T_t + S_t + \varepsilon_t$。
    *   不对序列进行差分，而是直接**对残差序列 $\varepsilon_t$ 拟合自回归模型**，以充分提取残余的相关信息：
        $\varepsilon_t = \phi_1 \varepsilon_{t-1} + \dots + \phi_p \varepsilon_{t-p} + a_t$
2. **确定性效应（趋势与季节）的拟合方法**
*   **趋势拟合**：有两种常用方法
    *   **方法一**：自变量为时间 $t$ 的多项式（或幂函数）：
        $$T_t = \beta_0 + \beta_1 t + \dots + \beta_k t^k + \varepsilon_t$$
    *   **方法二**：自变量为历史观察值的延迟项（如一阶延迟）：
        $$T_t = \beta_0 + \beta_1 x_{t-1} + \dots + \beta_k x_{t-k} + \varepsilon_t$$
*   **季节拟合**：有两种常用方法提取周期为 $m$ 的季节因子 $S_t$
    *   **方法一**：给定（计算）季节指数 $S_t$ 进行修正 (即提取确定性的固定季节因子)：
        $$S'_t = S_t \qquad$$
    *   **方法二**：建立具有周期跨度项的季节自回归模型：
        $$S_t = \alpha_0 + \alpha_1 x_{t-m} + \dots + \alpha_l x_{t-lm}$$
3. **残差序列的自相关检验**
    *   **原理**：如果回归模型拟合充分，残差之间应无相关性（$E(\varepsilon_t \varepsilon_{t-j})=0$）；若不充分则残差高度自相关。
    *   **Durbin-Watson (DW) 检验**：
        *   统计量公式：$DW = \frac{\sum_{t=2}^n (\varepsilon_t - \varepsilon_{t-1})^2}{\sum_{t=1}^n \varepsilon_t^2}$。
        *   与自相关系数的关系：$DW \approx 2(1-\rho)$。
        *   判定法则：DW 值在 $0 \sim d_L$ 为正相关；在 $d_U \sim 4-d_U$ 为不相关（关于 2 对称）；在 $4-d_L \sim 4$ 为负相关。中间区域为待定区。
    *   **DW 检验的缺陷与 Durbin h 检验**：
        *   缺陷：当回归因子中**包含延迟因变量（即历史数据 $X_{t-1}$ 作为解释变量）时**，残差序列的 DW 统计量会是有偏的。此时使用 DW 检验极易产生“残差序列正相关性不显著”的**误判**！
        *   对策：此时必须使用 **Durbin h (Dh) 检验**。统计量 $Dh \propto DW$，大样本下可通过判定 $Dh$ 拒绝无相关的原假设，以此得出序列高度正相关的正确结论。
4. **残差序列的拟合**

    在通过 DW / Dh 检验发现残差存在高度自相关后，我们必须对残差序列拟合自回归模型，以充分提取相关信息。完整的建模有三个标准步骤：
    1.  **确定自回归模型的阶数 (定阶)**：绘制残差序列的 ACF 和 PACF 图，根据截尾性或利用 AIC/SBC 信息准则，初步选定拟合阶数。
    2.  **参数估计**：通常采用极大似然估计法（或逐步回归法）估计出 $\phi_1, \phi_2 \dots$ 等参数。
    3.  **模型检验与最终口径确立**：最后写出 Auto-Regressive 模型的最终拟合方程组。
    * 注：建立好若干候选的确定性分离模型后，可以计算它们的 AIC/SBC 值，与直接差分拟合的 ARIMA 模型比较，选择信息准则最小者作为最终模型。

### 四、 异方差的性质与方差齐性变换（chapter4.一.1）
1. **异方差的定义、影响与直观诊断**
    *   **定义**：随机误差序列的方差随着时间的变化而变化，即 $Var(\varepsilon_t) = h(t)$。
    *   **影响（纳伪错误）**：忽视异方差的存在会导致残差的方差被严重低估，从而使得普通 OLS 参数显著性检验容易犯**纳伪错误（把本不显著的参数误判为显著）**，导致拟合精度严重受损。
    *   **直观诊断**：
        *   残差图：观察波动幅度是否随时间呈现“扇形（喇叭口）”扩散或收敛。
        *   **残差平方图**：因为均值为0时，$Var(\varepsilon_t) = E(\varepsilon_t^2)$，所以重点考察残差平方序列是否平稳，若呈趋势则存在异方差。
2. **方差齐性变换 (已知异方差函数的形式)**
    *   **使用场合**：序列显示出显著的异方差性，且方差与均值之间具有明确的函数关系 $\sigma_t^2 = h(\mu_t)$。
    *   **确定转换函数的推导过程（泰勒展开原理）**：
        *   设转换函数为 $g(X_t)$，将其在均值 $\mu_t$ 附近作一阶泰勒展开：$g(X_t) \approx g(\mu_t) + (X_t - \mu_t)g'(\mu_t)$。
        *   求转换后变量的方差：$Var[g(X_t)] \approx [g'(\mu_t)]^2 Var(X_t) = [g'(\mu_t)]^2 h(\mu_t)$。
        *   要使方差变为恒定常数，必须满足 $g'(\mu_t) = \frac{1}{\sqrt{h(\mu_t)}}$。
    *   **常用结论**：若假定标准差与均值成正比 $\sigma_t = \mu_t \implies h(\mu_t) = \mu_t^2$，解微分方程得 $g'(\mu_t) = 1/\mu_t$，积分即得著名的**对数变换法则** $g(X_t) = \log(X_t)$。

### 五、 条件异方差模型 (GARCH 族，未知异方差函数的形式)
如果不知异方差函数的具体形式，需要依靠残差信息的自相关关系来拟合异方差模型。

1. **ARCH 模型 (Autoregressive Conditional Heteroskedasticity)**
    *   **适用场合**：异方差函数主要表现为**短期**自相关过程。
    *   **模型结构**：通过直接构造**残差平方序列的自回归模型**来拟合条件方差。
        
        均值方程：$x_t = f(t, x_{t-1}, \dots) + \varepsilon_t$
        
        分布假定：$\varepsilon_t = \sqrt{h_t} e_t, \quad e_t \sim N(0,1)$
        方差方程：$h_t = \omega + \sum_{j=1}^q \lambda_j \varepsilon_{t-j}^2$
2. **GARCH 模型 (Generalized ARCH)**
    *   **适用场合**：当异方差函数表现为**长期**自相关时，ARCH 需要很高阶数。GARCH 通过引入历史的条件方差项 $h_{t-i}$（类似于 MA 项的变体）极大简化了参数量。
    *   **方差方程结构**：
        $h_t = \omega + \sum_{i=1}^p \eta_i h_{t-i} + \sum_{j=1}^q \lambda_j \varepsilon_{t-j}^2$
    *   **模型强约束条件**：
        1.  **参数非负**：保证方差恒正：$\omega \ge 0, \eta_i \ge 0, \lambda_j > 0$。
        2.  **参数有界（平稳性条件）**：冲击不会无限发散：$\sum \eta_i + \sum \lambda_j < 1$。
3. **GARCH 模型的四大变体**

    为了适应金融市场中比简单序列更为复杂的特征（如杠杆效应、风险溢价等），在基础的 $h_t = \omega + \sum \eta_i h_{t-i} + \sum \lambda_j \varepsilon_{t-j}^2$ 之后，衍生出了四大经典变体。

    *   **EGARCH 模型 (指数 GARCH)**：
        *   **核心改进**：通过对条件方差取自然对数 $\ln(h_t)$ 建模，直接解除了方差参数必须非负的严苛约束；同时引入了包含 $|e_{t-i}| - E(|e_{t-i}|)$ 和 $e_{t-i}$ 的不对称项，能有效捕捉金融市场中的**正负冲击非对称效应（即杠杆效应）**。
        *   **完整方程**：
            $$\begin{cases} 
            x_t = f(t, x_{t-1}, \dots) + \varepsilon_t \\
            \varepsilon_t = \sqrt{h_t} e_t \\
            \ln(h_t) = \omega + \sum_{i=1}^p \eta_i \ln(h_{t-i}) + \sum_{j=1}^q \lambda_j g(e_{t-j}) \\
            g(e_t) = \theta e_t + \gamma [|e_t| - E(|e_t|)]
            \end{cases}$$

    *   **IGARCH 模型 (单整 GARCH)**：
        *   **核心改进**：当 GARCH 模型的参数总和刚好达到边界（即 $\sum \eta_i + \sum \lambda_j = 1$）时，模型即为 IGARCH。它意味着序列中方差的波动具有“长记忆性”，过去的方差冲击对于未来的影响是永久性的（冲击不衰减）。
        *   **完整方程**：形式与标准 GARCH 完全一致，但附加了严格的单位根等式约束条件：
            $$\begin{cases} 
            h_t = \omega + \sum_{i=1}^p \eta_i h_{t-i} + \sum_{j=1}^q \lambda_j \varepsilon_{t-j}^2 \\
            s.t. \quad \sum_{i=1}^p \eta_i + \sum_{j=1}^q \lambda_j = 1 
            \end{cases}$$

    *   **GARCH-M 模型 (均值 GARCH)**：
        *   **核心改进**：将条件方差项（或标准差、对数方差）直接作为解释变量，引入到均值方程 $x_t$ 中。用于量化金融学中的“风险溢价”理论（高预期波动率/风险，必然要求高预期收益率的补偿）。
        *   **完整方程**：
            $$\begin{cases} 
            x_t = f(t, x_{t-1}, \dots) + \delta h_t + \varepsilon_t \\
            h_t = \omega + \sum_{i=1}^p \eta_i h_{t-i} + \sum_{j=1}^q \lambda_j \varepsilon_{t-j}^2
            \end{cases}$$
            *(注：此处的 $+\delta h_t$ 就是新增的风险溢价项，通常 $\delta > 0$)*

    *   **AR-GARCH 模型**：
        *   **核心改进**：对于短期自相关的残差，直接使用其前几期的序列波动来建立自回归均值方程，并将最终提取出的白噪声再投入 GARCH 方程进行异方差剥离。
        *   **完整方程**：
            $$\begin{cases} 
            x_t = f(t, x_{t-1}, \dots) + \varepsilon_t \\
            \varepsilon_t = \sum_{k=1}^m \beta_k \varepsilon_{t-k} + \nu_t \\
            \nu_t = \sqrt{h_t} e_t \\
            h_t = \omega + \sum_{i=1}^p \eta_i h_{t-i} + \sum_{j=1}^q \lambda_j \nu_{t-j}^2
            \end{cases}$$
4. **GARCH 模型的拟合与检验步骤**

    完整的 GARCH 建模遵循核心四步检验法：回归均值方程 $\to$ 残差自相关检验 (消除均值相关) $\to$ **异方差自相关性检验** $\to$ 正态性检验。具体的检验假设与统计量如下：
*   **残差自相关检验（消除均值方程相关性）**：
    *   **目的**：检验均值方程的残差 $\varepsilon_t$ 是否存在线性自相关。GARCH 模型要求投入方差方程的扰动项必须是“干净的”（无线性自相关的纯粹异方差聚集）。
    *   **检验方法**：同上文所属，使用 **DW 检验** 或 **Durbin h 检验**（若均值方程含延迟因变量，则必须用 Durbin h 检验）。
    *   **处理对策**：如果检验拒绝无相关的原假设，说明均值方程提取信息不充分。此时**必须对残差序列额外拟合自回归方程（如使用逐步回归法建立 $AR(m)$ 模型）**，提取出最终没有自相关的纯净残差项 $\nu_t$，然后再将其带入下一步进行 ARCH 效应检验。

*   **异方差自相关检验 (ARCH 效应检验)**：
    *   **假设条件**：$H_0: \rho_1 = \dots = \rho_q = 0$ （即残差平方序列无自相关，不存在 ARCH 效应）。
    *   **Portmanteau Q 检验**：利用 LB 统计量原理对残差平方序列做检验。
        *   统计量：$$Q(q) = n(n+2) \sum_{i=1}^q \frac{\hat{\rho}_i^2}{n-i}$$ （大样本下服从 $\chi^2(q)$ 分布）
        *   判定法则：若 $Q(q) \ge \chi^2_{1-\alpha}(q)$，则拒绝 $H_0$，证明序列存在 ARCH 效应。
    *   **LM 检验 (拉格朗日乘子检验)**：
        *   统计量：$$LM(q) = \frac{n}{2} \mathbf{W}' \mathbf{W}$$
            (大样本下 $LM(q) \sim \chi^2(q)$)
        *   判定法则：若 $LM(q) \ge \chi^2_{1-\alpha}(q)$，则拒绝 $H_0$，证明存在 ARCH 效应。

*   **最终残差正态性检验**：
    *   **假设条件**：$H_0: u_t \sim N(0,1)$ （即剥离条件异方差后的最终标准化残差服从标准正态分布）。
    *   **统计量**：利用最终标准化残差的偏度 $b_1$ 和峰度 $b_2$，构造统计量：
        $$T_n = n \left( \frac{b_1^2}{6} + \frac{(b_2 - 3)^2}{24} \right)$$
    *   **判定法则**：大样本下 $T_n \sim \chi^2(2)$。若不拒绝原假设（即 $T_n < \chi^2_{1-\alpha}(2)$ 或 $P > \alpha$），则说明最终残差满足正态性，GARCH 模型拟合完全充分，模型成立。
