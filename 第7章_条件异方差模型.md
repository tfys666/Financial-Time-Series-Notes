# 《金融时间序列》复习总结

## 第7章 条件异方差模型 (GARCH 模型)

### 一、 条件异方差模型产生的背景与意义

**1. 传统计量经济学（OLS）面临的挑战**
*   **同方差假定**：传统的普通最小二乘法 (OLS) 假定回归模型的扰动项 $u_t$ 的方差是一个常数（$Var(u_t) = \sigma^2$）。
*   **异方差的后果**：如果真实的残差存在异方差（如金融时间序列中的波动聚集性），虽然 OLS 估计的参数仍然是无偏且一致的，但它是**无效的（伪方差）**。参数估计量的标准误差会被严重低估或高估，导致 $t$ 检验和置信区间完全失效。

**2. 条件矩与无条件矩的核心区分**

条件异方差模型的基础建立在条件矩理论之上。为了解决异方差问题，必须严格从数学上区分“无条件矩”和“条件矩”。

*   **条件期望 (Conditional Mean)**
    *   条件期望是指在给定随机变量 $X$ 取值的条件下，$Y$ 的期望。它本身是关于 $X$ 的函数，即**条件期望是一个随机变量**。
    *   连续型公式：$E(Y|X) = \int y \cdot f(y|x) dy$
    *   离散型公式：$E(Y|X) = \sum y \cdot f(y|x)$
*   **迭代期望定理 (Law of Iterated Expectations)**
    *   对条件期望再求一次关于 $X$ 的期望，就等于无条件期望。
    *   公式推导：
        $$E(Y) = E_X[E(Y|X)] = \int \left[ \int y \cdot f(y|x) dy \right] f(x) dx$$
*   **条件方差 (Conditional Variance)**
    *   公式定义为 $Y$ 在给定 $X$ 下偏离其条件均值的平方的条件期望：
        $$Var(Y|X) = \int (Y - E(Y|X))^2 f(Y|X) dY$$
*   **方差分解公式 (Variance Decomposition)**
    *   这是引入 ARCH 模型最核心的数学工具。将随机变量的无条件方差（即总偏差平方和 $TSS$）进行拆解：
        $$Y - E(Y) = [Y - E(Y|X)] + [E(Y|X) - E(Y)]$$
    *   两边平方并求无条件期望。由于交叉项的期望为 0（即 $E[(Y - E(Y|X))(E(Y|X) - E(Y))] = 0$），总方差被完美拆分为两部分：
        1.  **误差平方和 (ESS)**：$E[(Y - E(Y|X))^2] = E_X[Var(Y|X)]$
        2.  **回归平方和 (RSS)**：$E[(E(Y|X) - E(Y))^2] = Var_X[E(Y|X)]$
    *   **结论**：
        $$Var(Y) = E_X[Var(Y|X)] + Var_X[E(Y|X)]$$
        *(即：无条件方差 = 条件方差的期望 + 条件期望的方差)*
*   **在 ARCH 模型中的应用意义**
    *   对于回归模型的残差 $u_t$，由于其条件均值为 0，即 $E(u_t | u_{t-1}) = 0$，所以“条件期望的方差”项为 0（$Var[E(u_t | u_{t-1})] = 0$）。
    *   代入方差分解公式，得到：
        $$Var(u_t) = E[Var(u_t | u_{t-1})] = E(\sigma_t^2)$$
    *   这一公式在理论上证明了：**金融序列的无条件方差可以是一个平稳的常数，但其条件方差（即局部的波动率 $\sigma_t^2$）却可以是随时间剧烈变化的随机变量**。

### 二、 ARCH 模型 (Autoregressive Conditional Heteroscedasticity)
1982年由 Engle 提出，核心思想是：残差序列 $u_t$ 本身是无序列相关的白噪声（条件均值为 0，无条件方差恒定），但其条件方差 $\sigma_t^2$ 却强烈依赖于过去残差的平方。它刻画了金融市场波动的聚集性。

**1. ARCH 模型的定义形式与假设**

完整的 ARCH 模型分为均值方程和方差方程：
*   **均值方程**（例如 AR(p) 形式）：
    $$y_t = b_0 + b_1 x_{1,t} + \dots + u_t$$
*   **白噪声假定与分布**：
    假设 $u_t = \sigma_t v_t$，且通常假定独立同分布创新项 $v_t \sim iid \ N(0,1)$。
    此时有：$E(u_t) = 0$ 且 $Cov(u_t, u_{t-\tau}) = 0 \ (\tau \neq 0)$。
*   **方差方程（条件异方差自回归项）**：
    残差的条件方差 $\sigma_t^2$ 被表示为前 $q$ 期残差平方项的线性组合（即 $ARCH(q)$）：
    $$\sigma_t^2 = Var(u_t | u_{t-1}, \dots, u_{t-q}) = \alpha_0 + \alpha_1 u_{t-1}^2 + \dots + \alpha_q u_{t-q}^2$$
    *(等价地，可写为 $u_t^2 = \alpha_0 + \alpha_1 u_{t-1}^2 + \dots + \alpha_q u_{t-q}^2 + w_t$，这里 $w_t$ 是服从 $E(w_t)=0$ 且无序列相关的纯净创新项)*

**2. ARCH(1) 模型的参数约束**

以最简单的一阶模型 $ARCH(1)$ 为例：$\sigma_t^2 = \alpha_0 + \alpha_1 u_{t-1}^2$。
必须施加两重限制才能保证方差的物理意义和序列平稳性：

*   **(1) 非负性约束**：方差恒正，故参数必须满足：
    $$\alpha_0 > 0, \quad \alpha_1 \ge 0$$
*   **(2) 二阶矩（无条件方差）的平稳性约束**：
    由方差分解可得，无条件方差等于条件方差的无条件期望。对方差方程两边求期望：
    $$E(u_t^2) = E(\sigma_t^2) = E(\alpha_0 + \alpha_1 u_{t-1}^2) = \alpha_0 + \alpha_1 E(u_{t-1}^2)$$
    因为在平稳序列中 $E(u_t^2) = E(u_{t-1}^2) = Var(u_t)$ 是一个恒定的常数，代入得：
    $$Var(u_t) = \alpha_0 + \alpha_1 Var(u_t)$$
    解得无条件方差为：
    $$\sigma_u^2 = Var(u_t) = \frac{\alpha_0}{1 - \alpha_1}$$
    由于方差必须大于 0 且不能无穷发散，所以参数必须满足平稳条件：**$0 \le \alpha_1 < 1$**。

**3. ARCH(1) 的四阶矩与“尖峰厚尾”**

金融数据普遍具有尖峰厚尾特性，而 ARCH 模型令人惊艳之处在于：即使假定最基础的随机项 $v_t \sim N(0,1)$ 是薄尾的，ARCH 过程本身依然会**内生出**比正态分布更厚的尾部。证明如下：

*   **步骤一：求条件四阶矩**
    
    已知 $u_t = \sigma_t v_t$，且 $v_t \sim N(0,1)$（对于标准正态分布，其四阶中心矩为 $3\sigma^4$）。
    故 $E(u_t^4 | u_{t-1}) = E(\sigma_t^4 v_t^4 | u_{t-1}) = \sigma_t^4 E(v_t^4) = 3\sigma_t^4$。
    代入方差方程 $\sigma_t^2 = \alpha_0 + \alpha_1 u_{t-1}^2$：
    $$E(u_t^4 | u_{t-1}) = 3(\alpha_0 + \alpha_1 u_{t-1}^2)^2 = 3(\alpha_0^2 + 2\alpha_0 \alpha_1 u_{t-1}^2 + \alpha_1^2 u_{t-1}^4)$$
*   **步骤二：求无条件四阶矩 $m_4$**
    
    根据迭代期望法则，对上式再次求无条件期望，设 $E(u_t^4) = m_4$：
    $$m_4 = E[E(u_t^4 | u_{t-1})] = 3E(\alpha_0^2 + 2\alpha_0 \alpha_1 u_{t-1}^2 + \alpha_1^2 u_{t-1}^4)$$
    在平稳假设下，$E(u_{t-1}^2) = \frac{\alpha_0}{1-\alpha_1}$ 且 $E(u_{t-1}^4) = m_4$，代入并展开：
    $$m_4 = 3\alpha_0^2 + \frac{6\alpha_0^2 \alpha_1}{1-\alpha_1} + 3\alpha_1^2 m_4$$
    合并含有 $m_4$ 的项并通分：
    $$m_4 = \frac{3\alpha_0^2(1-\alpha_1) + 6\alpha_0^2 \alpha_1}{(1-\alpha_1)(1-3\alpha_1^2)} = \frac{3\alpha_0^2(1+\alpha_1)}{(1-\alpha_1)(1-3\alpha_1^2)}$$
    为了确保分布的四阶矩存在且大于 0，必须满足更严苛的约束条件：
    $$1 - 3\alpha_1^2 > 0 \implies 0 \le \alpha_1 < \sqrt{\frac{1}{3}}$$
*   **步骤三：计算无条件峰度 $K$**
    
    峰度的定义为 $K = \frac{E(u_t^4)}{[E(u_t^2)]^2} = \frac{m_4}{(Var(u_t))^2}$：
    $$K = \frac{\frac{3\alpha_0^2(1+\alpha_1)}{(1-\alpha_1)(1-3\alpha_1^2)}}{\frac{\alpha_0^2}{(1-\alpha_1)^2}} = \frac{3\alpha_0^2(1+\alpha_1)(1-\alpha_1)^2}{\alpha_0^2(1-\alpha_1)(1-3\alpha_1^2)} = 3 \cdot \frac{1-\alpha_1^2}{1-3\alpha_1^2}$$
    因为 $\alpha_1^2 < 3\alpha_1^2 \implies 1-\alpha_1^2 > 1-3\alpha_1^2$，因此必然有：
    $$K > 3$$
    **结论**：由于无条件峰度恒大于正态分布的理论峰度 3，数学上严密证明了 ARCH(1) 残差序列具有重尾性。

**4. 基础 ARCH 效应检验与 ARCH 模型的局限**
*   **ARCH 的核心作用**：在实际应用中，由于纯粹的 ARCH($q$) 模型对参数（$\alpha_i$）的约束过于严苛，如今它主要不再作为最终拟合模型使用，而是作为一种检验手段（ARCH检验），用来测试残差序列是否具有条件异方差效应。
*   **ARCH 效应联合检验的逻辑**：
    *   首先通过普通的一元/多元回归（或 AR(n)）拟合均值方程，提取残差。
    *   构造辅助回归方程：基于 ARCH 模型的定义，将残差平方 $\hat{u}_t^2$ 对其自身滞后的 $q$ 阶序列回归：
        $$\hat{u}_t^2 = \beta_0 + \beta_1 \hat{u}_{t-1}^2 + \dots + \beta_q \hat{u}_{t-q}^2 + w_t$$
    *   原假设 $H_0: \beta_1 = \beta_2 = \dots = \beta_q = 0$。如果联合检验（如 F 检验或后文的 LM 检验）显著拒绝原假设，说明波动具有相关性，就证明了序列存在 ARCH 效应。
*   **局限与缺陷**：为了捕捉金融市场中长久的波动记忆效应，纯粹的 ARCH 模型需要极大阶数的 $q$。这会导致大量参数估计不显著或无法满足非负约束，从而催生了利用有限参数包含无限记忆的 GARCH 模型。

### 三、 GARCH 模型 (Generalized ARCH)
为了解决 ARCH 模型参数过多的问题，Bollerslev (1986) 提出了广义 ARCH 模型。GARCH 模型通过在等式右边引入条件方差自身的滞后项 $\sigma_{t-j}^2$（相当于 ARMA 中的 MA 项转化为 AR 形式的无穷阶展开），仅用极少的参数就能表达无穷阶的 ARCH 过程。

**1. GARCH 模型的定义与无穷阶 ARCH 等价性**

GARCH 模型通过在方差方程等式右边直接引入**条件方差自身的滞后项** $\sigma_{t-j}^2$，以极少的参数完美表达了无穷阶的 ARCH 过程。
*   **GARCH(p,q) 方差方程**：
    $$\sigma_t^2 = \alpha_0 + \sum_{i=1}^q \alpha_i u_{t-i}^2 + \sum_{j=1}^p \beta_j \sigma_{t-j}^2$$
    *(其中：$\alpha_i$ 为 ARCH 项，衡量近期外部冲击的大小；$\beta_j$ 为 GARCH 项，衡量历史波动记忆的长短)*
*   **与 $ARCH(\infty)$ 的等价性**：
    
    以最简单的 GARCH(1,1) 为例：$\sigma_t^2 = \alpha_0 + \alpha_1 u_{t-1}^2 + \beta_1 \sigma_{t-1}^2$。
    将 $\sigma_{t-1}^2 = \alpha_0 + \alpha_1 u_{t-2}^2 + \beta_1 \sigma_{t-2}^2$ 不断向后迭代代入：
    $$\begin{aligned} \sigma_t^2 &= \alpha_0 + \alpha_1 u_{t-1}^2 + \beta_1 (\alpha_0 + \alpha_1 u_{t-2}^2 + \beta_1 \sigma_{t-2}^2) \\ &= \alpha_0(1 + \beta_1) + \alpha_1 u_{t-1}^2 + \alpha_1 \beta_1 u_{t-2}^2 + \beta_1^2 \sigma_{t-2}^2 \\ &= \dots \\ &= \frac{\alpha_0}{1 - \beta_1} + \alpha_1 \sum_{j=1}^\infty \beta_1^{j-1} u_{t-j}^2 \end{aligned}$$
    **结论**：GARCH(1,1) 仅仅包含 3 个参数（$\alpha_0, \alpha_1, \beta_1$），就等价于一个拥有无穷多项、且权重以 $\beta_1$ 的几何级数衰减的 $ARCH(\infty)$ 模型！极大节约了自由度。

**2. 残差平方的 ARMA 表示与 GARCH 参数约束**

为了推导 GARCH 的无条件方差，我们引入一个纯净的白噪声扰动项 $w_t = u_t^2 - \sigma_t^2$（因为 $E_t(u_t^2) = \sigma_t^2$，所以 $E_t(w_t) = 0$）。
将 $\sigma_t^2 = u_t^2 - w_t$ 代入 GARCH(1,1) 方程：
$$u_t^2 - w_t = \alpha_0 + \alpha_1 u_{t-1}^2 + \beta_1 (u_{t-1}^2 - w_{t-1})$$
移项整理得到：
$$u_t^2 = \alpha_0 + (\alpha_1 + \beta_1) u_{t-1}^2 + w_t - \beta_1 w_{t-1}$$
**结论**：**GARCH(1,1) 模型的残差平方序列 $u_t^2$ 实际上服从一个 ARMA(1,1) 过程**！其中自回归系数为 $(\alpha_1 + \beta_1)$，滑动平均系数为 $-\beta_1$。

根据平稳 ARMA 过程求无条件均值的法则，对方程两边求无条件期望：
$$E(u_t^2) = \alpha_0 + (\alpha_1 + \beta_1) E(u_{t-1}^2) + E(w_t) - \beta_1 E(w_{t-1})$$
因为 $E(w_t)=0$，且平稳下 $E(u_t^2) = E(u_{t-1}^2) = \sigma_u^2$，解得**无条件方差**为：
$$\sigma_u^2 = \frac{\alpha_0}{1 - \alpha_1 - \beta_1}$$

为了保证方差的物理意义（恒正且有限收敛），GARCH(1,1) 的参数必须满足**严格约束条件**：
1.  **非负性约束**：$\alpha_0 > 0, \quad \alpha_1 \ge 0, \quad \beta_1 \ge 0$
2.  **平稳性（边界）约束**：$\alpha_1 + \beta_1 < 1$

*(注：同理可证，GARCH(1,1) 的无条件峰度 $K = \frac{3[1 - (\alpha_1+\beta_1)^2]}{1 - (\alpha_1+\beta_1)^2 - 2\alpha_1^2} > 3$，内生具有尖峰厚尾特征)*

**3. 极大似然估计 (MLE) 与三种残差分布假定**

由于金融序列的尾部往往比 GARCH 内生产生的还要厚，估计模型时通常需要利用极大似然法（使用 BHHH 算法迭代），并对残差 $u_t$ 或标准化残差 $v_t = u_t / \sigma_t$ 提出三种不同的概率分布假设，以此构建对数似然函数 $\ln L(\theta)$：

*   **正态分布 (Normal / Gaussian Distribution)**
最基础的假设，其对数似然函数为：
$$\ln L(\theta) = -\frac{T}{2} \ln(2\pi) - \frac{1}{2} \sum_{t=1}^T \ln(\sigma_t^2) - \frac{1}{2} \sum_{t=1}^T \frac{(y_t - x'_t \gamma)^2}{\sigma_t^2}$$

*   **学生 t-分布 (Student's t-distribution)**
为了更精确地描述更宽的尾部特征，采用自由度为 $k$ ($k>2$) 的 t-分布。当 $k \to \infty$ 时，趋近于正态分布。
其对数似然函数为（利用 Gamma 函数 $\Gamma(\cdot)$）：
$$\ln L(\theta) = T \ln \left[ \frac{\Gamma(\frac{k+1}{2})}{\sqrt{\pi(k-2)}\Gamma(\frac{k}{2})} \right] - \frac{1}{2} \sum_{t=1}^T \ln(\sigma_t^2) - \frac{k+1}{2} \sum_{t=1}^T \ln \left[ 1 + \frac{(y_t - x'_t \gamma)^2}{\sigma_t^2(k-2)} \right]$$

*   **广义误差分布 (GED, Generalized Error Distribution)**
通过控制参数 $r > 0$ 调节尾部厚度。当 $r=2$ 时，GED 退化为正态分布；$r < 2$ 时比正态分布尾巴厚。
其对数似然函数为：
$$\ln L(\theta) = -\frac{T}{2} \ln \left[ \frac{\Gamma(1/r)^3}{\Gamma(3/r)(r/2)^2} \right] - \frac{1}{2} \sum_{t=1}^T \ln(\sigma_t^2) - \sum_{t=1}^T \left[ \frac{\Gamma(3/r)(y_t - x'_t \gamma)^2}{\sigma_t^2 \Gamma(1/r)} \right]^{r/2}$$


**4. GARCH 的滞后阶数选择与模型评价**

在确定 $p$ 和 $q$ 时，通常依据信息准则越小越好的原则挑选最优秀的模型：
*   **AIC 准则**：$AIC = -2l/T + 2k/T$
*   **SC (Schwarz / BIC) 准则**：$SC = -2l/T + (k \ln T)/T$

*(其中 $l$ 为对数似然值，$T$ 为样本量，$k$ 为参数个数。SC 准则对多余参数的惩罚更重)*

**5. GARCH 条件方差预测**

通过 $t$ 时刻的已知样本信息，推断样本外的未来方差。对于 GARCH(1,1) 模型：
*   **1步向前预测**（直接代入已知数据）：
    $$\hat{\sigma}_{t+1}^2 = \alpha_0 + \alpha_1 u_t^2 + \beta_1 \sigma_t^2$$
*   **2步向前预测**：
    预测 $t+2$ 期的方差时，由于 $t+1$ 期的残差平方 $u_{t+1}^2$ 是未知的，必须用其条件期望 $E_{t}(u_{t+1}^2) = \sigma_{t+1}^2$ 来替代：
    $$E_t(\sigma_{t+2}^2) = \alpha_0 + \alpha_1 E_t(u_{t+1}^2) + \beta_1 E_t(\sigma_{t+1}^2) = \alpha_0 + (\alpha_1 + \beta_1) \hat{\sigma}_{t+1}^2$$
*   **$n$ 步向前预测**（递推通式）：
    $$E_t(\sigma_{t+n}^2) = \alpha_0 + (\alpha_1 + \beta_1) E_t(\sigma_{t+n-1}^2)$$
*   **均值回归特性（预测的极限）**：
    由于约束条件 $\alpha_1 + \beta_1 < 1$，这是一个公比小于 1 的几何级数。当预测步数 $n \to \infty$ 时，多步预测条件方差必然**收敛于无条件方差**：
    $$\lim_{n \to \infty} E_t(\sigma_{t+n}^2) = \frac{\alpha_0}{1 - \alpha_1 - \beta_1}$$

### 四、 ARCH 效应的检验方法
判断一个时间序列是否需要用 GARCH 模型拟合，首先要检验其均值方程的残差是否存在 ARCH 效应（即条件异方差性）。

**1. ARCH LM 检验 (Lagrange Multiplier Test)**
*   **步骤**：
    1.  用 OLS 估计均值方程，提取残差 $\hat{u}_t$。
    2.  运行辅助回归方程（将残差平方对其滞后项回归）：
        $$\hat{u}_t^2 = \beta_0 + \beta_1 \hat{u}_{t-1}^2 + \dots + \beta_q \hat{u}_{t-q}^2 + \varepsilon_t$$
    3.  原假设 $H_0: \beta_1 = \beta_2 = \dots = \beta_q = 0$。
*   **检验统计量**：
    *   **$F$ 统计量**：检验所有系数联合显著性的省略变量检验。
    *   **$TR^2$ 统计量 (LM 统计量)**：$T$ 为观测样本数，$R^2$ 为辅助回归的拟合优度。在大样本下，$TR^2 \sim \chi^2(q)$。若 P 值小于显著性水平，则拒绝原假设，序列存在 ARCH 效应。

**2. 残差平方相关图 (Correlogram of Squared Residuals)**

直接考察残差平方序列 $\hat{u}_t^2$ 的 ACF 和 PACF，计算 Ljung-Box Q 统计量。如果存在 ARCH 效应，自相关系数会显著不为 0，且 Q 统计量的 P 值显著。

### 五、 非对称 GARCH 模型 (杠杆效应)
资本市场中通常存在“杠杆效应”：**资产价格下跌（利空/坏消息）带来的波动率增加，要显著大于等量价格上涨（利好/好消息）带来的波动率增加**。标准 GARCH 模型中由于使用的是残差平方 $u_{t-1}^2$，正负符号被抹除了，无法体现这种非对称性。

**1. TARCH 模型 (Threshold ARCH)**
通过引入虚拟变量 $d_{t-1}$ 来区分正负冲击的门限模型：
$$\sigma_t^2 = \omega + \alpha u_{t-1}^2 + \gamma d_{t-1} u_{t-1}^2 + \beta \sigma_{t-1}^2$$
*   **判别逻辑**：
    *   当 $u_{t-1} < 0$（坏消息）时，$d_{t-1} = 1$。冲击力度为 $\alpha + \gamma$。
    *   当 $u_{t-1} \ge 0$（好消息）时，$d_{t-1} = 0$。冲击力度仅为 $\alpha$。
*   **杠杆效应判定**：如果估计出的 $\gamma > 0$，说明坏消息的冲击显著大于好消息，存在杠杆效应。

**2. EGARCH 模型 (Exponential GARCH)**
直接对条件方差的自然对数 $\ln(\sigma_t^2)$ 建模，这样不需要对参数施加任何非负性约束：
$$\ln(\sigma_t^2) = \omega + \beta \ln(\sigma_{t-1}^2) + \alpha \left| \frac{u_{t-1}}{\sigma_{t-1}} \right| + \gamma \frac{u_{t-1}}{\sigma_{t-1}}$$
*   **判别逻辑**：
    *   当 $u_{t-1} > 0$（好消息）时，绝对值打开为正，冲击项为 $(\alpha + \gamma) \frac{u_{t-1}}{\sigma_{t-1}}$。
    *   当 $u_{t-1} < 0$（坏消息）时，绝对值打开为负，冲击项为 $(\alpha - \gamma) \frac{|u_{t-1}|}{\sigma_{t-1}}$。
*   **杠杆效应判定**：如果估计出的 **$\gamma < 0$**，则坏消息带来的乘数 $(\alpha - \gamma)$ 大于好消息的乘数 $(\alpha + \gamma)$，说明存在杠杆效应。

**3. PARCH 模型 (Power ARCH)**
将拟合对象从条件方差 $\sigma_t^2$ 拓展到了条件标准差的任意次幂 $\delta$：
$$\sigma_t^\delta = \omega + \sum_{j=1}^p \beta_j \sigma_{t-j}^\delta + \sum_{i=1}^q \alpha_i (|u_{t-i}| - \gamma_i u_{t-i})^\delta$$
*   **特点**：幂参数 $\delta > 0$ 是从数据中估计出来的，而非指定的。$\gamma_i$ 捕捉非对称效应。当 $\delta=2$ 且 $\gamma_i=0$ 时，退化为标准 GARCH 模型。

### 六、 其他 GARCH 衍生模型

**1. IGARCH 模型 (Integrated GARCH)**
*   **定义**：如果限定 GARCH 方差方程中的所有的 ARCH 项和 GARCH 项**系数之和等于 1**（且通常去掉常数项 $\omega$）：
    $$\sum_{i=1}^q \alpha_i + \sum_{j=1}^p \beta_j = 1$$
*   **金融意义**：被称为单整 GARCH 模型。它意味着方差的冲击具有无限的记忆力（方差是一个随机游走过程），当前的波动冲击对未来预期的影响是持久的，永远不会衰减。

**2. ARCH-M 模型 (ARCH-in-mean / 均值 GARCH)**
*   **定义**：将条件方差 $\sigma_t^2$（或条件标准差 $\sigma_t$、对数方差 $\ln(\sigma_t^2)$）作为额外的解释变量，**引入到均值方程中**。
*   **模型方程**（以引入标准差为例）：
    $$y_t = x_t'\gamma + \rho \sigma_t + u_t$$
*   **金融意义**：完美契合了金融学中的“风险溢价 (Risk Premium)”理论——高预期风险要求高预期收益率进行补偿。如果估计出的系数 $\rho > 0$ 且显著，说明市场中的预期风险每增加一个百分点，就会导致收益率相应地增加 $\rho$ 个百分点。
