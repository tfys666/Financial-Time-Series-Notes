# 《金融时间序列》复习总结

## 第三章 线性平稳时间序列分析

### 方法性工具

**1. 延迟算子与差分运算**
*   **延迟算子 $B$**：定义 $B X_t = X_{t-1}$。
    *   性质：$B^0 = 1$；$B c = c$（$c$ 为常数）；$B(X_t \pm Y_t) = B X_t \pm B Y_t$；$B^n X_t = X_{t-n}$。
*   **差分运算 $\nabla$**：
    *   **一阶差分**：$\nabla X_t = X_t - X_{t-1} = (1-B)X_t$。
    *   **$p$ 阶差分**：$\nabla^p X_t = \nabla^{p-1} X_t - \nabla^{p-1} X_{t-1} = (1-B)^p X_t$。
    *   **$k$ 步差分**：$\nabla_k X_t = X_t - X_{t-k} = (1-B^k)X_t$。
*   **线性差分方程及其求解**：用来描述时间序列模型的底层数学结构。
    *   **二阶常系数线性差分方程的求解**
        标准形式：$y_{t+2} + a y_{t+1} + b y_t = f(t)$ （其中 $b \neq 0$）。
        *   **齐次方程**：$y_{t+2} + a y_{t+1} + b y_t = 0$。特征方程：$\lambda^2 + a\lambda + b = 0$，设 $\Delta = a^2 - 4b$。
            1. $\Delta > 0$（相异实根 $\lambda_1, \lambda_2$）：通解 $y_t = C_1 \lambda_1^t + C_2 \lambda_2^t$。
            2. $\Delta = 0$（相等实根 $\lambda_1 = \lambda_2 = -a/2$）：通解 $y_t = (C_1 + C_2 t) \lambda_1^t$。
            3. $\Delta < 0$（共轭复根 $\alpha \pm i\beta$）：设极坐标 $r = \sqrt{\alpha^2+\beta^2} = \sqrt{b}$，$\tan\omega = -\frac{\sqrt{4b-a^2}}{a}$。通解 $y_t = r^t(C_1 \cos t\omega + C_2 \sin t\omega)$。
        *   **非齐次方程特解 $y_t^*$ 的设定方法**：
            1. **当 $f(t) = c$ (常数) 时**：
                * 设特解形式为 $y_t^* = k t^s$。
                * 若 $1+a+b \neq 0$：取特解 $y_t^* = k = \frac{c}{1+a+b}$。
                * 若 $1+a+b = 0$ 且 $1-b \neq 0$ ($a \neq -2$)：取特解 $y_t^* = kt = \frac{c}{a+2}t$。
                * 若 $1+a+b = 0$ 且 $1-b = 0$ ($a = -2$)：取特解 $y_t^* = kt^2 = \frac{c}{2}t^2$。
            2. **当 $f(t) = c q^t$ (指数形式，$c, q$ 为常数且 $q \neq 1$) 时**：
                * 设特解形式为 $y_t^* = k t^s q^t$。
                * 若 $q^2 + aq + b \neq 0$：取 $s=0$，得特解 $y_t^* = k q^t = \frac{cq^t}{q^2 + aq + b}$。
                * 若 $q^2 + aq + b = 0$ 且 $2q + a \neq 0$：取 $s=1$，得特解 $y_t^* = kt q^{t-1} = \frac{ct ^{qt-1}}{2q+a}$。
                * 若 $q^2 + aq + b = 0$ 且 $2q + a = 0$：取 $s=2$，得特解 $y_t^* = kt^2 q^{t-2} = \frac{ct^{qt-1}}{4q+a}$。
            3. **当 $f(t) = c t^n$ (多项式形式，$c$ 为常数) 时**：
                * 设特解形式为 $y_t^* = t^s (B_0 + B_1 t + \cdots + B_n t^n)$，其中待定系数通过代入原方程比较同次项系数求得。
                * 若 $1+a+b \neq 0$：取 $s = 0$。
                * 若 $1+a+b = 0$ 且 $a \neq -2$：取 $s = 1$。
                * 若 $1+a+b = 0$ 且 $a = -2$：取 $s = 2$。
    *   **一般齐次线性差分方程（$p$ 阶推广）**：$z_t + a_1 z_{t-1} + a_2 z_{t-2} + \cdots + a_p z_{t-p} = 0$。
        *   **特征方程**：$\lambda^p + a_1\lambda^{p-1} + a_2\lambda^{p-2} + \cdots + a_p = 0$。其根称为特征根 $\lambda_1, \lambda_2, \cdots, \lambda_p$。特征根在单位圆内（即 $|\lambda| < 1$）是时间序列平稳性的核心判断标准。
        *   **齐次方程通解的表达式**（根据特征根分三种情况）：
            1. **全不相等实数根**（$\lambda_i \neq \lambda_j$）：$z_t = c_1 \lambda_1^t + c_2 \lambda_2^t + \cdots + c_p \lambda_p^t$。
            2. **有相等实数根**（如 $\lambda_1$ 为 $d$ 重根）：$z_t = (c_1 + c_2 t + \cdots + c_d t^{d-1}) \lambda_1^t + c_{d+1} \lambda_{d+1}^t + \cdots + c_p \lambda_p^t$。
            3. **复根场合**（如 $\lambda_{1,2} = \alpha \pm i\beta$）：化为三角形式 $\cos \omega, \sin \omega$，极径 $r = \sqrt{\alpha^2 + \beta^2}$。复根部分的解为 $r^t(C_1 \cos t\omega + C_2 \sin t\omega)$。
    *   **非齐次线性差分方程**：$z_t + a_1 z_{t-1} + a_2 z_{t-2} + \cdots + a_p z_{t-p} = h(t)$。
        *   **求解原理**：非齐次方程的通解 $z_t'$ = 该方程的任意一个**特解 $z_t^*$** + 对应齐次方程的**通解 $z_t$**。

**2. 线性过程与因果性 (Causality)**
*   **线性过程**：如果时间序列能表示为 $X_t = \sum_{j=-\infty}^{\infty} G_j \varepsilon_{t-j}$（其中 $\{\varepsilon_t\}$ 是白噪声，$\sum_{j=-\infty}^{\infty} G_j^2 < \infty$），则 $\{X_t\}$ 称为线性过程。线性过程一定是平稳序列。
*   **因果性**：若 $G_j = 0$ 对所有 $j < 0$ 成立，即 $X_t = \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$（当前的 $X_t$ 只受当前和过去的扰动影响，不受未来扰动影响），则该系统是因果的。记作 $X_t = G(B)\varepsilon_t$，其中 $G(B) = \sum_{j=0}^\infty G_j B^j$ 为 **Green 函数**（或传递函数）。
    *   **因果平稳条件**：$G(B)$ 在 $|B| \le 1$ 绝对收敛 $\Leftrightarrow$ $G(B)$ 的所有根（零点）都在单位圆外。
*   **逆转形式 (Invertibility)**：理论研究和实际问题处理时，通常需要用 $t$ 时刻及 $t$ 时刻以前的历史观测值 $X_{t-j} \ (j=0,1,\cdots)$ 来表示当前的白噪声扰动 $\varepsilon_t$。
    *   逆转形式表示为：$\varepsilon_t = \sum_{j=0}^{\infty} I_j X_{t-j} = I(B)X_t$，其中 $I(B) = G^{-1}(B)$ 称为逆函数。
*   **线性过程因果性与可逆性的等价条件**：
    *   已知线性过程 $X_t = \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$，且 $\sum_{j=0}^{\infty} |G_j| < \infty$。
    *   **结论**：如果多项式 $G(B) = \sum_{j=0}^\infty G_j B^j \neq 0$ 对所有 $|B| \le 1$ 成立（即 $G(B)$ 的**零点都在单位圆外**），则存在逆转形式：
        $\varepsilon_t = \sum_{j=0}^{\infty} I_j X_{t-j} = I(B)X_t$
        并且满足 $\sum_{j=0}^{\infty} |I_j| < \infty$，且 $I(B) = \sum_{j=0}^\infty I_j B^j \neq 0$ 对所有 $|B| \le 1$ 成立（即 $I(B)$ 的零点也全在单位圆外）。
    *   **定理意义**：该定理说明，对于一个因果的平稳线性过程，$G(B)$ 的零点都在单位圆外，与 $I(B)$ 的零点都在单位圆外（可逆性）是完全等价的。这也构成了后来判定 ARMA 模型是否平稳且可逆的理论基础。
*   **线性滤波及其交换律 (Linear Filters)**：
    *   **定义**：将一个输入序列 $x_t$ 通过系数为 $h_j$ 的线性算子转化为输出序列 $y_t$ 的过程称为线性滤波，记作 $y_t = \sum_j h_j x_{t-j} = h(B)x_t$。这里的 $h(B)$ 称为**滤波器**。
    *   **级联滤波器 (Cascaded Filters)**：如果一个序列先后通过两个滤波器 $h^{(1)}(B)$ 和 $h^{(2)}(B)$，那么总的滤波效果是两者的乘积，即 $y_t = h^{(2)}(B)[h^{(1)}(B)x_t] = h^{(2)}(B)h^{(1)}(B)x_t$。
    *   **交换律**：因为延迟算子多项式乘法满足交换律，即 $h^{(1)}(B)h^{(2)}(B) = h^{(2)}(B)h^{(1)}(B)$。这意味着**无论以何种顺序让序列通过这几个线性滤波器，最终的输出序列是完全相同的**。
    *   **线性滤波的逆 (Inverse Filters)**：如果存在另一个滤波器 $h^{-1}(B)$ 使得 $h^{-1}(B)[h(B)x_t] = x_t$，即 $h^{-1}(B)h(B) = 1$，则称 $h^{-1}(B)$ 为 $h(B)$ 的逆滤波器。这实质上就是 $AR$ 模型和 $MA$ 模型相互转化的数学工具。

### ARMA模型

**3. 自回归模型(Auto Regression Model) AR(p)**
*   **定义**：当前值不仅与前 $p$ 期观测值有关，还受当前白噪声扰动。
    *   **非中心化模型**：$X_t = c + \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t$
    *   **中心化变换**：均值 $\mu = \frac{c}{1-\phi_1-\cdots-\phi_p}$，令 $Y_t = X_t - \mu$ 即可中心化。
    *   **中心化模型**：$\Phi(B)X_t = \varepsilon_t$，其中 $\Phi(B) = 1 - \phi_1 B - \cdots - \phi_p B^p$ 为自回归系数多项式。
*   **平稳性判别**：
    *   **特征根判别**：对应齐次差分方程的 $p$ 个特征根 $\lambda_i$ 全部在**单位圆内**（$|\lambda_i| < 1$）。
        *   **多项式判别**：等价于自回归系数多项式 $\Phi(B)=0$ 的根全在**单位圆外**。（原因在于自回归系数多项式和对应齐次差分方程的特征多项式的根互为倒数）
    *   **平稳域判别**：序列平稳 ⇔ 所有特征根的模小于 1 ⇔ 参数落在平稳域内。对于低阶自回归模型用平稳域的方法判别模型的平稳性通常更为简便。
        *   对于 AR(1)， $X_t = \phi_1 X_{t-1} + \varepsilon_t$，平稳域为 $|\phi_1| < 1$。
        *   对于 AR(2)， $X_t = \phi_1 X_{t-1} + \phi_2 X_{t-2} + \varepsilon_t$，平稳域为 $\phi_1 + \phi_2 < 1, \phi_2 - \phi_1 < 1, |\phi_2| < 1$。
*   **均值**：
    由于 $\{X_t\}$ 是平稳序列，且白噪声 $\{\varepsilon_t\}$ 的均值为 $0$，对非中心化模型 $X_t = \phi_0 + \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t$ 两边取期望，得：$EX_t = \mu = \frac{\phi_0}{1 - \phi_1 - \cdots - \phi_p}$

*   **Green 函数**：
    $$X_t = \frac{\varepsilon_t}{\Phi(B)} = \sum_{i=1}^p \frac{k_i}{1 - \lambda_i B} \varepsilon_t = \sum_{i=1}^p \sum_{j=0}^{\infty} k_i (\lambda_i B)^j \varepsilon_t = \sum_{j=0}^{\infty} \left(\sum_{i=1}^p k_i \lambda_i^j\right) \varepsilon_{t-j} \triangleq \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$$
    其中常数序列 $\{G_j, j=1,2,\cdots\}$ 称为 **Green 函数**。即：任意的AR(p)都可以表示成一组纯随机序列的线性组合。

    *   **AR(p) 的 Green 函数递推公式**（待定系数法）：
        *   原理：$\begin{cases} \Phi(B)X_t = \varepsilon_t \\ X_t = G(B)\varepsilon_t \end{cases} \Rightarrow \Phi(B)G(B)\varepsilon_t = \varepsilon_t$
        *   递推公式结果：
            * $G_0 = 1$
            * $G_j = \sum_{k=1}^j \phi_k' G_{j-k}, \quad j = 1, 2, \cdots$，其中 $\phi_k' = \begin{cases} \phi_k, & k \le p \\ 0, & k > p \end{cases}$

*   **方差**：
    利用 Green 函数得到的平稳 AR 模型的传递形式 $X_t = \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$，两边求方差得：$Var(X_t) = \sum_{j=0}^{\infty} G_j^2 Var(\varepsilon_{t-j}) = \sum_{j=0}^{\infty} G_j^2 \sigma_\varepsilon^2 \quad (G_j \text{为Green函数})$

*   **协方差**：
    
    平稳 AR(p) 模型：
    $X_t = \phi_1 X_{t-1} + \phi_2 X_{t-2} + \cdots + \phi_p X_{t-p} + \varepsilon_t$
    
    首先，在模型两边同乘 $\varepsilon_t$，并取期望得：
    $E(X_t \varepsilon_t) = E(\varepsilon_t^2) = \sigma_\varepsilon^2$

    其次，在模型两边同乘 $X_t$，并取期望得：
    $\gamma_0 = E(X_t X_t) = \phi_1 \gamma_1 + \phi_2 \gamma_2 + \cdots + \phi_p \gamma_p + \sigma_\varepsilon^2$
    
    最后，在平稳 AR(p) 模型两边同乘 $X_{t-k} \ (k \ge 1)$，再求期望得：
    $E(X_t X_{t-k}) = \phi_1 E(X_{t-1} X_{t-k}) + \phi_2 E(X_{t-2} X_{t-k}) + \cdots + \phi_p E(X_{t-p} X_{t-k}) + E(\varepsilon_t X_{t-k})$
    
    根据 $E(\varepsilon_t X_{t-k}) = 0, \ \forall k \ge 1$，得到协方差函数的递推公式：$$\gamma_k = \phi_1 \gamma_{k-1} + \phi_2 \gamma_{k-2} + \cdots + \phi_p \gamma_{k-p}, \quad \forall k \ge 1$$

*   **自相关系数 (ACF) 的性质：拖尾性**
    *   **定义**：$\rho_k = \frac{\gamma_k}{\gamma_0}$。
    *   **Yule-Walker 方程**：
        由 AR(p) 模型的自协方差递推公式 $\gamma_k = \phi_1 \gamma_{k-1} + \cdots + \phi_p \gamma_{k-p} \ (k > 0)$，等式两边同除以 $\gamma_0$，即得 Yule-Walker 方程：
        $$\rho_k = \phi_1 \rho_{k-1} + \phi_2 \rho_{k-2} + \cdots + \phi_p \rho_{k-p}, \quad \forall k > 0$$
    *   **注**：
        Yule-Walker 方程实质上是一个**齐次差分方程**。设其通解为 $\rho_k = \sum_{i=1}^p c_i \lambda_i^k$，其中 $\lambda_i$ 是 $p$ 阶特征根。
        因为模型平稳，要求所有特征根都在单位圆内（即 $|\lambda_i| < 1$），这导致 $\rho_k$ 会随着 $k \to \infty$ 而逐渐趋于 $0$。
        但由于 $c_1, \cdots, c_p$ 不可能恒等于 $0$，因此 $\rho_k$ 的衰减呈现为**复指数衰减**（负指数单调收敛、正负相间衰减、“伪周期”震荡、不规则衰减），绝对不会在某阶后骤然恒等于 0。这种逐渐衰减的性质称为**拖尾性**。

*   **偏自相关系数 (PACF) 的性质：$p$ 步截尾性**
    *   **背景**：
        延迟 $k$ 阶自相关系数 $\rho_k$ 衡量的并不是 $X_t$ 与 $X_{t-k}$ 之间单纯的相关关系，它还受到中间 $k-1$ 个变量 $X_{t-1}, X_{t-2}, \cdots, X_{t-k+1}$ 的影响。
        **延迟 $k$ 阶偏自相关系数**就是在给定中间 $k-1$ 个随机变量的条件下，或者说，在剔除了中间 $k-1$ 个随机变量的干扰之后，$X_{t-k}$ 对 $X_t$ 影响的相关度量。
    *   **定义**：
        用数学语言描述就是条件期望相关的表达式：
        $$\phi_{kk} = \rho(X_t, X_{t-k} | X_{t-1}, \cdots, X_{t-k+1}) = \frac{E[(X_t - \hat{E}X_t)(X_{t-k} - \hat{E}X_{t-k}) | X_{t-1}, \cdots, X_{t-k+1}]}{E[(X_t - \hat{E}X_t)^2 | X_{t-1}, \cdots, X_{t-k+1}]}$$
        其中 $\hat{E}X_t$ 和 $\hat{E}X_{t-k}$ 是基于中间变量的条件期望。
    *   **最小方差估计的视角**：
        对于零均值平稳序列 $\{X_t\}$，考虑用 $X_{t-k}, \cdots, X_{t-1}$ 对 $X_t$ 进行线性最小方差估计，即选择系数使得残差方差达到极小：
        $$E\left(X_t - \sum_{j=1}^k \phi_{kj} X_{t-j}\right)^2 \to \min$$
        其中，第 $k$ 阶自回归模型拟合中的最后一个系数 $\phi_{kk}$ 就是 $X_t$ 的偏自相关系数。它衡量了在模型中已经包含了前 $k-1$ 个滞后项后，**再增加一期滞后 $X_{t-k}$ 所能额外增加的模型的解释能力**。
    *   **Yule-Walker 方程组与 PACF 的计算**：
        为了求出 $\phi_{kk}$，用过去的序列值对 $X_t$ 做 $k$ 阶自回归拟合：$$X_t = \phi_{k1}X_{t-1} + \phi_{k2}X_{t-2} + \cdots + \phi_{kk}X_{t-k} + \varepsilon_t$$
        在等号两边同乘 $X_{t-l} \ (l=1,2,\cdots,k)$，并取期望，再同除以 $\gamma_0$，可得到由 $k$ 个方程构成的 Yule-Walker 方程组：
        $$\rho_l = \phi_{k1} \rho_{l-1} + \phi_{k2} \rho_{l-2} + \cdots + \phi_{kk} \rho_{l-k}, \quad \forall l = 1, 2, \cdots, k$$
        将其写成矩阵形式：
        $$
        \begin{pmatrix} 1 & \rho_1 & \cdots & \rho_{k-1} \\ \rho_1 & 1 & \cdots & \rho_{k-2} \\ \vdots & \vdots & \ddots & \vdots \\ \rho_{k-1} & \rho_{k-2} & \cdots & 1 \end{pmatrix}
        \begin{pmatrix} \phi_{k1} \\ \phi_{k2} \\ \vdots \\ \phi_{kk} \end{pmatrix}
        = \begin{pmatrix} \rho_1 \\ \rho_2 \\ \vdots \\ \rho_k \end{pmatrix}
        $$
        根据 Cramer 法则，偏自相关系数 $\phi_{kk}$ 可以解为：
        $$\phi_{kk} = \frac{D_k}{D}$$
        其中 $D$ 是系数矩阵（自相关矩阵）的行列式，$D_k$ 是将 $D$ 的最后一列替换为等号右侧常数向量后的行列式。
    *   **Levinson-Durbin 递推公式**：
        由于求解高阶矩阵的行列式非常困难，实际计算中通常使用 Levinson-Durbin 递推公式来求解偏自相关系数 $\phi_{kk}$：
        $$\phi_{kk} = \frac{\rho_k - \sum_{j=1}^{k-1} \phi_{k-1, j} \rho_{k-j}}{1 - \sum_{j=1}^{k-1} \phi_{k-1, j} \rho_j}$$
        其中 $\phi_{kj} = \phi_{k-1, j} - \phi_{kk} \phi_{k-1, k-j}, \quad j = 1, 2, \cdots, k-1$。
    *   **AR(p) 模型的 $p$ 步截尾性证明逻辑**：
        
        对于平稳 AR(p) 序列，当滞后阶数 $k > p$ 时，由于其自相关系数完全受 Yule-Walker 方程 $\rho_k = \phi_1\rho_{k-1} + \cdots + \phi_p\rho_{k-p}$ 的支配，这就意味着行列式 $D_k$ 的最后一列可以被前面的 $p$ 列完全线性表出。
        根据线性代数性质，行列式列向量线性相关时，行列式的值等于 0。
        因此，当 $k > p$ 时，必定有 $D_k = 0 \Rightarrow \phi_{kk} = 0$。
        这就是 AR(p) 模型偏自相关系数在 $p$ 阶之后**严格截尾**（等于 0）的数学本质。
        
        定理：零均值平稳序列 $\{X_t\}$ 为 AR(p) 序列的充要条件是 $\{X_t\}$ 的偏自相关系数 $p$ 步截尾。

**4. 移动平均模型(Moving Average Model) MA(q)**
*   **定义**：系统在 $t$ 时刻的响应 $X_t$ 仅与其以前时刻的响应 $X_{t-j}$ 有关，而与其以前时刻进入系统的扰动 $\varepsilon_{t-j}$ 无关。
    *   **非中心化模型**：$X_t = \mu +\varepsilon_t - \theta_1 \varepsilon_{t-1} - \cdots - \theta_q \varepsilon_{t-q}$
    *   **中心化模型**：$X_t = \varepsilon_t - \theta_1 \varepsilon_{t-1} - \cdots - \theta_q \varepsilon_{t-q}$
    *   **算子表示**：$X_t = \Theta(B)\varepsilon_t$，其中 $\Theta(B) = 1 - \theta_1 B - \cdots - \theta_q B^q$ 为移动平均系数多项式。
*   **平稳性**：由于 MA(q) 模型是白噪声的有限线性组合（即不存在无穷阶历史扰动的累积），它**天然满足平稳性条件**，一定是一个平稳模型。
*   **常数均值**：模型两边求期望得 $E(X_t) = \mu$。
*   **常数方差**：$Var(X_t) = Var(\varepsilon_t - \theta_1 \varepsilon_{t-1} - \cdots - \theta_q \varepsilon_{t-q}) = (1 + \theta_1^2 + \theta_2^2 + \cdots + \theta_q^2)\sigma_\varepsilon^2$
*   **自协方差系数的 $q$ 步截尾性**：
        
    在模型两边同乘 $X_{t-k}$ 并取期望：
        $$\gamma_k = E[(X_t - \mu)(X_{t-k} - \mu)] = E[(\varepsilon_t - \theta_1\varepsilon_{t-1} - \cdots - \theta_q\varepsilon_{t-q})(\varepsilon_{t-k} - \theta_1\varepsilon_{t-k-1} - \cdots - \theta_q\varepsilon_{t-k-q})]$$
        只有当两个扰动项的下标相同时期望才为 $\sigma_\varepsilon^2$，否则为 0。因此：
        $$
        \gamma_k = \begin{cases}
        (1 + \theta_1^2 + \cdots + \theta_q^2)\sigma_\varepsilon^2, & k = 0 \\
        (-\theta_k + \theta_1\theta_{k+1} + \cdots + \theta_{q-k}\theta_q)\sigma_\varepsilon^2, & 1 \le k \le q \\
        0, & k > q
        \end{cases}
        $$
        自协方差只与滞后阶数 $k$ 相关，且在 $k > q$ 时骤然降为 0，即表现出 $q$ 步截尾性。
*   **自相关系数 (ACF) 与偏自相关系数 (PACF)**
    *   **自相关系数 ($q$ 步截尾)**：
        $$\rho_k = \frac{\gamma_k}{\gamma_0} = \begin{cases} 1, & k=0 \\ \frac{-\theta_k + \sum_{i=1}^{q-k} \theta_i \theta_{k+i}}{1 + \theta_1^2 + \cdots + \theta_q^2}, & 1 \le k \le q \\ 0, & k > q \end{cases}$$
        MA(q) 的 ACF 表现为在 $q$ 步之后截断等于 0。
    *   **偏自相关系数 (拖尾)**：
        对于 MA(q) 模型，由于其逆函数（将 $X_t$ 转化为无限项 AR 形式）具有无穷多个系数 $I_j$ 不恒等于 0，偏自相关系数呈现出不断衰减但不骤然等于 0 的**拖尾性**。（详细证明略）
*   **MA 模型的可逆性 (Invertibility)**
    *   **自相关系数的不唯一性**：
        不同的 MA 模型可能具有完全相同的自相关系数和偏自相关系数。例如，$X_t = \varepsilon_t - 2\varepsilon_{t-1}$ 和 $X_t = \varepsilon_t - 0.5\varepsilon_{t-1}$ 的 ACF 是相同的（$\rho_1 = \frac{-2}{1+4} = \frac{-0.5}{1+0.25} = -0.4$）。
        为了使一个自相关系数序列能够**唯一对应**一个 MA 模型，我们必须引入可逆性概念。
    *   **可逆的定义**：若一个 MA 模型能够表示为**收敛**的**无穷阶 AR 模型**形式，那么该 MA 模型称为**可逆 MA 模型**。（一个自相关系数列唯一对应一个可逆MA模型，解决了上述不唯一的问题）
    *   **MA(q) 模型的可逆条件**：
        *   MA(q) 模型特征方程的特征根都在**单位圆内**，或移动平滑系数多项式 $\Theta(B) = 0$ 的根都在**单位圆外**。
        *   例如：对于 MA(1) $X_t = \varepsilon_t - \theta_1\varepsilon_{t-1}$，特征方程为 $1 - \theta_1 B = 0$，根 $B = \frac{1}{\theta_1}$。要使其在单位圆外，必须 $|B| > 1$，即**平稳可逆条件为 $|\theta_1| < 1$**。
        *   如果 $|\theta_1| > 1$，模型不可逆。但我们总可以找到另一个参数为 $\frac{1}{\theta_1}$ 的可逆 MA 模型与其拥有相同的自相关结构。
    *   MA(q) 模型的可逆性和 AR(p) 模型的平稳性是对偶概念。

*   **逆函数的递推公式（待定系数法）**
    *   原理：由 $X_t = \Theta(B)\varepsilon_t$ 转化为可逆形式 $\varepsilon_t = I(B)X_t = \sum_{j=0}^{\infty} I_j X_{t-j}$（假设，参考AR模型，类似）。
        $$\Rightarrow I(B)\Theta(B)\varepsilon_t = \varepsilon_t \Rightarrow I(B)\Theta(B) = 1$$
    *   将级数展开：$(I_0 + I_1 B + I_2 B^2 + \cdots)(1 - \theta_1 B - \theta_2 B^2 - \cdots - \theta_q B^q) = 1$。
    *   比较同次幂系数，得到**逆函数 $I_j$ 的递推公式**：
        *   $I_0 = 1$
        *   $I_j = \sum_{k=1}^j \theta_k' I_{j-k}, \quad (j \ge 1)$
        *   其中，当 $k \le q$ 时 $\theta_k' = \theta_k$；当 $k > q$ 时 $\theta_k' = 0$。

**5. 自回归移动平均模型(Auto Regression Moving Average Model) ARMA(p, q)**
*   **定义**
    *   **背景**：一个系统，如果它在 $t$ 时刻的响应 $X_t$ 不仅与其以前时刻的响应有关（AR特性），而且还与其以前时刻进入系统的扰动存在着一定的相关关系（MA特性），那么这个系统就是自回归移动平均系统，相应的模型记作 ARMA 模型。
    *   **定义**：具有如下结构的模型称为自回归移动平均模型，简记为 $ARMA(p, q)$：
        $$X_t = c + \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t - \theta_1 \varepsilon_{t-1} - \cdots - \theta_q \varepsilon_{t-q}$$
        其中 $\phi_p \neq 0, \theta_q \neq 0$；$\{\varepsilon_t\}$ 为白噪声序列。
    *   **算子表示**：引进延迟算子 $B$，非中心化的 ARMA(p,q) 模型可简记为 $\Phi(B)X_t = c + \Theta(B)\varepsilon_t$；特别当 $c=0$ 时，称为中心化模型：
        $$\Phi(B)X_t = \Theta(B)\varepsilon_t$$
        其中：
        * $p$ 阶自回归系数多项式：$\Phi(B) = 1 - \phi_1 B - \phi_2 B^2 - \cdots - \phi_p B^p$
        * $q$ 阶移动平均系数多项式：$\Theta(B) = 1 - \theta_1 B - \theta_2 B^2 - \cdots - \theta_q B^q$

*   **平稳域与可逆域**
    
    对于 ARMA(p,q) 模型来说，只有**平稳且可逆**才是有实际意义的。
    *   **平稳条件**：
        完全由其自回归部分 $\Phi(B)$ 决定。平稳域为：$\{\phi_i : \Phi(B)=0 \text{ 的根都在单位圆外}\}$。如果系统具有平稳性，说明系统对某一时刻进入的扰动的记忆逐渐衰减，时间越远，影响作用越小。
    *   **可逆条件**：
        完全由其移动平均部分 $\Theta(B)$ 决定。可逆域为：$\{\theta_j : \Theta(B)=0 \text{ 的根都在单位圆外}\}$。可逆性表示某一时刻的系统响应对后继时刻的响应的影响呈递减状态。
    *   **平稳可逆域**：同时满足上述两个条件的参数集合。即：$\{\phi_i,\theta_j : \Phi(B)=0 \text{ 和 } \Theta(B)=0 \text{ 的根都在单位圆外}\}$。

*   **传递形式与逆转形式**
    *   **传递形式 (Wold 分解)**：将 ARMA 转化为 $MA(\infty)$ 模型：
        $$X_t = \Phi^{-1}(B)\Theta(B)\varepsilon_t = \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$$
        使用待定系数法展开 $\Phi(B) \sum_{j=0}^{\infty} G_j \varepsilon_{t-j} = \Theta(B)\varepsilon_t$，可得 Green 函数 $G_k$ 的递推公式：
        $$G_0 = 1$$
        $$G_k = \sum_{j=1}^k \phi_j' G_{k-j} - \theta_k', \quad k \ge 1$$
        其中：$\phi_j' = \begin{cases} \phi_j, & 1 \le j \le p \\ 0, & j > p \end{cases}, \quad \theta_k' = \begin{cases} \theta_k, & 1 \le k \le q \\ 0, & k > q \end{cases}$

    *   **逆转形式**：将 ARMA 转化为 $AR(\infty)$ 模型：
        $$\varepsilon_t = \Theta^{-1}(B)\Phi(B)X_t = \sum_{j=0}^{\infty} I_j X_{t-j}$$
        使用待定系数法展开 $\Theta(B) \sum_{j=0}^{\infty} I_j B^j = \Phi(B)$，可得逆函数 $I_k$ 的递推公式：
        $$I_0 = 1$$
        $$I_k = \sum_{j=1}^k \theta_j' I_{k-j} - \phi_k', \quad k \ge 1$$
        其中：$\theta_j' = \begin{cases} \theta_j, & 1 \le j \le q \\ 0, & j > q \end{cases}, \quad \phi_k' = \begin{cases} \phi_k, & 1 \le k \le p \\ 0, & k > p \end{cases}$

*   **统计性质**
    *   **均值**：$EX_t = \mu = \frac{c}{1 - \phi_1 - \cdots - \phi_p}$。
    *   **方差**：借助传递形式 $X_t = \sum_{j=0}^{\infty} G_j \varepsilon_{t-j}$，得 $Var(X_t) = \sum_{j=0}^{\infty} G_j^2 \sigma_\varepsilon^2$。
    *   **自协方差**：借助传递形式，$\gamma_k = E(X_{t+k} X_t) = E[(\sum_{i=0}^{\infty} G_i \varepsilon_{t+k-i})(\sum_{j=0}^{\infty} G_j \varepsilon_{t-j})] = \sum_{j=0}^{\infty} G_j G_{j+k} \sigma_\varepsilon^2$。
    *   **自相关系数**：$\rho_k = \frac{\gamma_k}{\gamma_0} = \frac{\sum_{j=0}^{\infty} G_j G_{j+k}}{\sum_{j=0}^{\infty} G_j^2}$。

*   **相关性特征**
    *   **自相关系数 (ACF)**：**拖尾**。
        类似于 AR(p) 模型，由于自回归多项式 $\Phi(B)$ 的存在，ARMA 模型的 ACF 在滞后足够大的阶数后受齐次差分方程控制，呈现出指数衰减或正负交替衰减，不会在某阶后突然截断为零。
    *   **偏自相关系数 (PACF)**：**拖尾**。
        定理：设 $\{X_t\}$ 为 MA(q) 序列或者 ARMA(p,q) 序列，则 $\{X_t\}$ 的偏自相关系数必定拖尾。
        （原因在于 MA 部分使得 ARMA 可以转化为无限项的 AR 模型，因此回归系数 $\phi_{kk}$ 永远不会等于零）。
*   **三种模型转换**：
    当 AR、MA 和 ARMA 都具有平稳可逆性时，它们可以相互转换：
    1. $AR(p) \rightarrow MA(\infty)$ （平稳性保证）
    2. $MA(q) \rightarrow AR(\infty)$ （可逆性保证）
    3. $ARMA(p, q) \rightarrow AR(\infty)$ 或 $MA(\infty)$ （平稳且可逆保证）

### 平稳序列建模

**6. 模型识别**

模型识别的核心工具是样本自相关函数 (ACF) 和偏自相关函数 (PACF) 的图形特征。（模型识别之前还应该进行单位根检验，先确保平稳性，这在第二章中已经讲解）
*   **计算样本相关系数**：
    *   **样本自相关系数 (Sample ACF)**：
        $\hat{\rho}_k = \frac{\sum_{t=1}^{n-k} (x_t - \bar{x})(x_{t+k} - \bar{x})}{\sum_{t=1}^n (x_t - \bar{x})^2}$
    *   **样本偏自相关系数 (Sample PACF)**：
        由 Cramer 法则解 Yule-Walker 方程得到：
        $\hat{\phi}_{kk} = \frac{\hat{D}_k}{\hat{D}}$
*   **判定表**：
    | 选择模型 | 样本自相关系数 $\hat{\rho}_k$ | 样本偏自相关系数 $\hat{\phi}_{kk}$ |
    | :--- | :--- | :--- |
    | **AR(p)** | 拖尾 | **$p$ 阶截尾** |
    | **MA(q)** | **$q$ 阶截尾** | 拖尾 |
    | **ARMA(p, q)** | 拖尾 | 拖尾 |
*   **模型定阶的经验方法与 95% 置信区间**：
    在实际样本中，理论上的“截尾”并不会真正变为绝对的 0，而是在 0 附近作小值随机波动。根据 Bartlett 和 Quenouille 定理，$\hat{\rho}_k$ 和 $\hat{\phi}_{kk}$ 在大样本下近似服从 $N(0, 1/n)$。
    因此，判断截尾性的经验法则是看其是否落在 $95\%$ 的置信区间内：
    $$Pr\left(-\frac{2}{\sqrt{n}} \le \hat{\rho}_k \le \frac{2}{\sqrt{n}}\right) \ge 0.95，Pr\left(-\frac{2}{\sqrt{n}} \le \hat{\phi}_{kk} \le \frac{2}{\sqrt{n}}\right) \ge 0.95$$
    如果样本系数在最初的 $d$ 阶明显超出两倍标准差范围 $\pm \frac{2}{\sqrt{n}}$，而后突然衰减且几乎 95% 的系数都落入两倍标准差范围内波动，即可视为 $d$ 阶截尾。

**7. 参数估计**

当确定了模型形式（如 ARMA(p,q)）后，需要对模型中的 $p+q+2$ 个未知参数 $(\phi_1, \dots, \phi_p, \theta_1, \dots, \theta_q, \mu, \sigma_\varepsilon^2)$ 进行估计。参数估计的时候都没有考虑到平稳性或者可逆性的约束，所以参数估计后都要算一下相应特征方程的特征根是不是在单位圆内。
*   **矩估计 (Method of Moments)**：
    *   **原理**：用样本的一阶矩、二阶矩（即样本均值 $\hat{\mu} = \bar{x}$、样本方差 $\hat{\gamma}_0$ 和样本自相关系数 $\hat{\rho}_k$）去代替总体的相应矩，代入 Yule-Walker 等总体矩方程中，反解出未知参数的估计值。
    *   **求解实例**：
        *   **AR(2) 的矩估计**：利用Yule-Walker 方程，由 $\rho_1 = \phi_1 + \phi_2\rho_1$ 及 $\rho_2 = \phi_1\rho_1 + \phi_2$。将 $\rho$ 替换为 $\hat{\rho}$ 后，解得：
            $$\hat{\phi}_1 = \frac{\hat{\rho}_1(1 - \hat{\rho}_2)}{1 - \hat{\rho}_1^2}, \quad \hat{\phi}_2 = \frac{\hat{\rho}_2 - \hat{\rho}_1^2}{1 - \hat{\rho}_1^2}$$
        *   **MA(1) 的矩估计**：由协方差的递推，相除有 $\rho_1 = \frac{-\theta_1}{1+\theta_1^2}$ 得到 $\theta_1^2 + \frac{1}{\rho_1}\theta_1 + 1 = 0$。为了满足可逆性条件 $|\theta_1| < 1$，解得：
            $$\hat{\theta}_1 = \frac{-1 + \sqrt{1 - 4\hat{\rho}_1^2}}{2\hat{\rho}_1}$$
            （要求 $|\hat{\rho}_1| \le 0.5$，否则无法求得实数解）。
        *   **方差 $\sigma_\varepsilon^2$ 的矩估计**：通过计算拟合序列 $\hat{x}_t$ 与观测值 $x_t$ 的残差 $\varepsilon_t = x_t - \hat{x}_t$，由于残差独立同分布，方差矩估计为 $\hat{\sigma}_\varepsilon^2 = \frac{1}{n} \sum_{t=1}^n \varepsilon_t^2$。
    *   **评价**：思想简单直观，计算量小，不需要假设总体分布；但**信息浪费严重**（只用到 $p+q$ 个相关系数，高阶信息被忽略），估计精度差。实际中通常仅用作极大似然或最小二乘法的**迭代初始值**。

*   **极大似然估计 (Maximum Likelihood Estimation)**：
    *   **原理**：认为样本来自使该样本出现概率最大的总体。通常假设序列服从零均值的多元正态分布，利用协方差矩阵 $\Omega$ 构造似然函数：
        $$L(\beta; x_1, \dots, x_n) = \max\{ p(x_1, \dots, x_n) ; \beta \}$$
        对数似然函数为：$l(\beta) = -\frac{n}{2} \ln(2\pi) - \frac{n}{2}\ln(\sigma_\varepsilon^2) - \frac{1}{2}\ln|\Omega| - \frac{S(\beta)}{2\sigma_\varepsilon^2}$
        其中 $S(\beta) = X^T \Omega^{-1} X$。
    *   **求解**：通过令偏导 $\frac{\partial l}{\partial \beta} = 0$ 得到似然方程。由于 $S(\beta)$ 和 $\ln|\Omega|$ 都不是 $\beta$ 的显式表达式，似然方程是由 $p+q+1$ 个超越方程构成的，通常需要利用**复杂的迭代算法**（如牛顿-拉夫逊法）求数值解。
    *   **评价**：充分利用了每一个观察值提供的信息，具有**估计的一致性、渐近正态性和渐近有效性**，估计精度高；缺点是必须事先假定序列服从正态分布，且计算复杂（一般依赖软件）。

*   **最小二乘估计 (Least Squares Estimation)**：
    *   **原理与特征**：基本准则是使残差平方和 $Q(\beta) = \min \sum_{t=1}^n \varepsilon_t^2$ 达到最小。对于 AR 模型，残差是参数的线性函数；但对于包含 MA 部分的模型（MA、ARMA），由于历史随机扰动 $\varepsilon_{t-1}, \varepsilon_{t-2}, \cdots$ 是**不可观测的**，因此 $Q(\beta)$ 通常不是参数的显式函数。
    *   **转化为 $AR(\infty)$ 形式**：求解时往往需要利用逆转形式，将模型转化为无限阶自回归：
        $$\varepsilon_t = \frac{\Phi(B)}{\Theta(B)}x_t = x_t - \sum_{i=1}^t \pi_i x_{t-i}$$
        正因如此，未知参数的最小二乘估计值通常必须**借助迭代法**求出。
    *   **条件最小二乘估计 (Conditional Least Squares)**：
        为了能进行上述迭代计算并截断无穷历史项，必须给出初始条件。其核心是**假定过去未观测到的序列值等于零（即 $x_t = 0, \ t \le 0$）**。在这个假定条件下优化的最小二乘估计即为条件最小二乘估计：
        $$Q(\hat{\beta}) = \min \sum_{t=1}^n \left( x_t - \sum_{i=1}^t \pi_i x_{t-i} \right)^2$$
    *   **评价**：原理简单，方法普适，且充分利用了观察值信息，**估计精度高**。

**8. 模型检验**
*   **模型的显著性检验（残差检验）**：
    *   **目的**：检验模型对信息的提取是否充分（即模型是否有效）。一个好的拟合模型应该提取完几乎所有的相关信息，剩下的**残差序列必须是纯随机的白噪声**。
    *   **假设条件**：$H_0: \rho_1 = \rho_2 = \cdots = \rho_m = 0$ （残差序列为白噪声）。
    *   **统计量**：通常采用 **LB 统计量 (Ljung-Box)**：$LB = n(n+2) \sum_{k=1}^m \frac{\hat{\rho}_k^2}{n-k} \sim \chi^2(m-p-q)$。
    *   **结论**：若计算的 P 值 $> \alpha$ (如 $0.05$)，则接受 $H_0$，说明残差为白噪声，**模型显著有效**。
*   **参数的显著性检验**：
    *   **目的**：检验每一个未知参数是否显著非零，剔除不显著的参数，使**模型结构最精简**。
    *   **假设条件**：$H_0: \beta_j = 0 \leftrightarrow H_1: \beta_j \neq 0, \quad \forall 1 \le j \le m$。
    *   **检验统计量**：
        采用 t 统计量，其计算公式为：
        $$T = \frac{\hat{\beta}_j}{\sqrt{a_{jj} \frac{Q(\beta)}{n-m}}} \sim t(n-m)$$
        其中 $\hat{\beta}_j$ 是待检验参数的估计值；$a_{jj}$ 是在估计过程中得到的参数协方差矩阵的第 $j$ 个主对角元；$Q(\beta)$ 是残差平方和；$n-m$ 是自由度（$n$ 为样本容量，$m$ 为模型中待估参数的个数）。
    *   **判定原则**：若该统计量的绝对值大于 $t_{\alpha/2}(n-m)$ 或 P 值 $< 0.05$ 时，拒绝原假设，表明该参数显著非零，应该保留在模型中。

**9. 模型优化**

当同一个序列拟合了多个都显著有效的模型时，需要用信息准则确定“相对最优模型”。好的模型应该是拟合精度（似然函数大）和未知参数个数（自变量少，避免过拟合）的综合最优配置。
*   **AIC 准则 (Akaike Information Criterion)**：
    *   定义式：$AIC = -2 \ln(\text{模型的极大似然函数值}) + 2 \times (\text{模型中未知参数个数})$
    *   在正态分布的假设下，其计算公式为：
        *   中心化 ARMA(p,q) 模型（无常数项，未知参数有 $p+q+1$ 个）：
        $$AIC = n \ln(\hat{\sigma}_\varepsilon^2) + 2(p + q + 1)$$
        *   非中心化 ARMA(p,q) 模型（有常数项，未知参数有 $p+q+2$ 个）：
        $$AIC = n \ln(\hat{\sigma}_\varepsilon^2) + 2(p + q + 2)$$
    *   缺陷：惩罚权重固定为 2，导致在样本容量 $n \to \infty$ 时，AIC 选择的模型不收敛于真实模型，通常倾向于选择**参数过多**的模型。

*   **SBC / BIC 准则 (Schwarz Bayesian Criterion)**：
    *   定义式：$SBC = -2 \ln(\text{极大似然函数值}) + (\ln n) \times (\text{模型中未知参数个数})$
    *   在正态分布的假设下，其计算公式为：
        *   中心化 ARMA(p,q) 模型的 SBC 函数：
        $$SBC = n \ln(\hat{\sigma}_\varepsilon^2) + (\ln n)(p + q + 1)$$
        *   非中心化 ARMA(p,q) 模型的 SBC 函数：
        $$SBC = n \ln(\hat{\sigma}_\varepsilon^2) + (\ln n)(p + q + 2)$$
    *   改进：将未知参数个数的惩罚权重由常数 2 改为了**样本容量的对数函数 $\ln(n)$**。理论证明 SBC 准则是最优模型真实阶数的相合估计，对复杂模型的惩罚比 AIC 更重。

*   **判定原则**：在所有通过检验的模型中，**使得 AIC 或 SBC 函数值达到最小的模型为相对最优模型**。对于同一模型比较时，通常只需看 $\hat{\sigma}_\varepsilon^2$ 和参数个数构成的后一个计算式。

### 序列预测

**10. 最优预测原则**

要在已知历史信息 $X$（可能是向量）的条件下预测未来序列值 $Y$，在**均方误差 (Mean Square Error)** 意义下的**最优预测是条件期望**：$\hat{Y} = E(Y|X)$。
*   **数学证明**：
    对于任意预测函数 $f(X)$，预测均方误差为：
    $$E[Y - f(X)]^2 = E[(Y - E(Y|X)) + (E(Y|X) - f(X))]^2$$
    展开后交叉项期望为零，故有：
    $$E[Y - f(X)]^2 = E[Y - E(Y|X)]^2 + E[E(Y|X) - f(X)]^2 \ge E[Y - E(Y|X)]^2$$
    当且仅当 $f(X) = E(Y|X)$ 时，等号成立，均方误差达到最小。

**11. 序列的预测分解与误差分析**

假设平稳序列可以用传递形式表示为 $X_t = \sum_{i=0}^\infty G_i \varepsilon_{t-i}$。
我们要在 $t$ 时刻预测向前 $l$ 步（“向前” = 顺着时间方向走 = 未来）的序列值 $X_{t+l}$：
$$X_{t+l} = \sum_{i=0}^\infty G_i \varepsilon_{t+l-i} = (G_0 \varepsilon_{t+l} + \dots + G_{l-1} \varepsilon_{t+1}) + (G_l \varepsilon_t + G_{l+1} \varepsilon_{t-1} + \dots)$$
*   **预测值 $\hat{x}_t(l)$**：
    因为 $\varepsilon_{t+1}, \dots, \varepsilon_{t+l}$ 是未来时刻的白噪声扰动，在 $t$ 时刻其条件期望为 0。所以 $t$ 时刻对 $X_{t+l}$ 的 $l$ 步预测值（条件期望/最优预测）为：
    $$\hat{x}_t(l) = E(X_{t+l} | x_t, x_{t-1}, \dots) = G_l \varepsilon_t + G_{l+1} \varepsilon_{t-1} + \dots$$
*   **预测误差 $e_t(l)$**：
    预测误差等于真实值减去预测值，即序列分解的第一部分：
    $$e_t(l) = X_{t+l} - \hat{x}_t(l) = G_0 \varepsilon_{t+l} + G_1 \varepsilon_{t+l-1} + \dots + G_{l-1} \varepsilon_{t+1}$$
    *   期望：$E[e_t(l)] = 0$ （因为每个 $\varepsilon$ 的期望都是 0）
    *   方差：$Var[e_t(l)] = (G_0^2 + G_1^2 + \dots + G_{l-1}^2)\sigma_\varepsilon^2 = \left( 1 + \sum_{i=1}^{l-1} G_i^2 \right) \sigma_\varepsilon^2$ （其中 $G_0 = 1$）。

**12. 各类平稳序列的预测公式**

预测时遵循客观逻辑约定：若时间下标 $t+j \le t$，则使用**真实的已知观测值 $X_{t+j}$ 或提取的残差 $\varepsilon_{t+j}$**；若 $t+j > t$，则使用**条件期望推导出的预测值 $\hat{x}_t(j)$**，并且未来白噪声的期望 $E(\varepsilon_{t+j}) = 0$。

*   **AR(p) 序列的预测**：
    *   **预测值计算**：模型为 $X_{t+l} = \phi_1 X_{t+l-1} + \cdots + \phi_p X_{t+l-p} + \varepsilon_{t+l}$
        ，两边取 $t$ 时刻的条件期望，得：$$\hat{x}_t(l) = \phi_1 \hat{x}_t(l-1) + \cdots + \phi_p \hat{x}_t(l-p)$$
    *   **预测方差**：利用 AR(p) 传递形式的 Green 函数 $G_j$，得 $Var[e_t(l)] = \left( 1 + G_1^2 + \dots + G_{l-1}^2 \right) \sigma_\varepsilon^2$。
    *   **预测的 95% 置信区间**：
        假设白噪声服从正态分布，向后 $l$ 步的未来真实值 $X_{t+l}$ 以 95% 的置信度落在区间：
        $$\hat{x}_t(l) \pm 1.96 \cdot \sigma_\varepsilon \cdot \sqrt{1 + G_1^2 + \cdots + G_{l-1}^2}$$

*   **MA(q) 序列的预测**：
    由于 MA 模型只包含前 $q$ 期的随机扰动，当预测步数 $l > q$ 时，过去的扰动将完全不再发挥作用。
    *   **预测值计算**：
        $$\hat{x}_t(l) = \begin{cases} \mu - \sum_{i=l}^q \theta_i \varepsilon_{t+l-i}, & l \le q \\ \mu, & l > q \end{cases}$$
    *   **预测方差**：MA(q) 的传递系数即为 $-\theta_i$：
        $$Var[e_t(l)] = \begin{cases} \left( 1 + \theta_1^2 + \dots + \theta_{l-1}^2 \right) \sigma_\varepsilon^2, & l \le q \\ \left( 1 + \theta_1^2 + \dots + \theta_q^2 \right) \sigma_\varepsilon^2, & l > q \end{cases}$$
        注意：当 $l>q$ 后，预测方差恒定等于序列的无条件方差 $(1 + \theta_1^2 + \dots + \theta_q^2)\sigma_\varepsilon^2$。

*   **ARMA(p,q) 序列的预测**：
    结合两者的截尾特性（主要是 MA 部分在 $q$ 步后的消失）：
    *   **预测值计算**：
        $$\hat{x}_t(l) = \begin{cases} \phi_1 \hat{x}_t(l-1) + \cdots + \phi_p \hat{x}_t(l-p) - \sum_{i=l}^q \theta_i \varepsilon_{t+l-i}, & l \le q \\ \phi_1 \hat{x}_t(l-1) + \cdots + \phi_p \hat{x}_t(l-p), & l > q \end{cases}$$
    *   **预测方差**：同样由递推所得的 Green 函数（参见前面 ARMA 模型部分）决定，$Var[e_t(l)] = (1 + G_1^2 + \dots + G_{l-1}^2)\sigma_\varepsilon^2$。

*   **预测的 95% 置信区间**：
    假设白噪声服从正态分布，则向后 $l$ 步的未来真实值 $X_{t+l}$ 以 95% 的置信度落在区间：
    $$\hat{x}_t(l) \pm 1.96 \sqrt{Var[e_t(l)]}$$

*   **长期特性与局限性**：
    由于 ARMA 模型是基于短期相关性的，随着预测步长 $l$ 的增加：
    1.  **预测值收敛**：预测值 $\hat{x}_t(l)$ 最终都会收敛到序列的无条件均值 $\mu$（中心化序列收敛到 $0$）。
    2.  **预测方差收敛**：预测误差方差 $Var[e_t(l)]$ 最终都会收敛且逐渐扩大至序列的无条件方差 $\gamma_0$。
    
    结论：**ARMA 模型仅适合做短期预测**。在长期预测中，模型能提供的信息只剩下均值。对于长记忆序列，应考虑使用 ARFIMA 模型（略）。
    *   **模型特征**：引入了分数阶差分参数 $d$（通常 $0 < d < 0.5$）。相比于 ARIMA 中 $d$ 必须为整数，ARFIMA 能够更灵活、更精确地捕捉金融时间序列（如波动率数据）中长期的缓慢衰减特征（长记忆性）。

**13. 修正预测**

当有新的真实观测值到来时，不需要重新估计全部模型参数，而是利用新信息修正原有预测值以提高精度。
*   **基础概念**：
    在 $t$ 时刻，我们对 $X_{t+l}$ 有一个旧的预测值 $\hat{x}_t(l)$。
*   **新获得 1 个观察值 $x_{t+1}$**：
    此时时刻推进到了 $t+1$，我们需要对 $X_{t+l}$（即向前的第 $l-1$ 步）进行新的预测。
    新获取的真实扰动信息为：$\varepsilon_{t+1} = x_{t+1} - \hat{x}_t(1)$。
    *   **修正预测值**：$\hat{x}_{t+1}(l-1) = \hat{x}_t(l) + G_{l-1}\varepsilon_{t+1}$。
    *   **修正预测误差**：$e_{t+1}(l-1) = e_t(l) - G_{l-1}\varepsilon_{t+1}$。
    *   **修正预测方差**：$Var[e_{t+1}(l-1)] = Var[e_t(l)] - G_{l-1}^2 \sigma_\varepsilon^2$。（方差变小，精度提高）。
*   **一般情况：新获得 $p$ 个观察值 $x_{t+1}, \dots, x_{t+p}$**：
    *   **修正预测值**：$\hat{x}_{t+p}(l-p) = \hat{x}_t(l) + G_{l-1}\varepsilon_{t+1} + \dots + G_{l-p}\varepsilon_{t+p}$。
    *   **修正预测误差**：$e_{t+p}(l-p) = e_t(l) - G_{l-1}\varepsilon_{t+1} - \dots - G_{l-p}\varepsilon_{t+p} = G_0 \varepsilon_{t+l} + G_1 \varepsilon_{t+l-1} + \dots + G_{l-p-1} \varepsilon_{t+p+1}$。
    *   **修正预测方差**：$Var[e_{t+p}(l-p)] = \left( 1 + G_1^2 + \dots + G_{l-p-1}^2 \right) \sigma_\varepsilon^2$。
