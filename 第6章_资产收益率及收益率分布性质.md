# 《金融时间序列》复习总结

## 第6章 资产收益率及收益率分布性质

### 一、 资产收益率的概念与计算
本章开篇从最基础的金融收益率定义出发，系统梳理了收益率的各种计算方式及其内在关联。

**1. 简单收益率 (Simple Return)**
*   **单期简单毛收益率**：$1 + R_t = \frac{P_t}{P_{t-1}}$
*   **单期简单净收益率**：$R_t = \frac{P_t - P_{t-1}}{P_{t-1}}$
*   **多期简单收益率**：持有 $k$ 期的简单净收益率为 $R_t[k] = \frac{P_t - P_{t-k}}{P_{t-k}}$。
    多期毛收益率等于单期毛收益率的连乘：$1 + R_t[k] = \prod_{j=0}^{k-1}(1 + R_{t-j})$。
*   **年化收益率**：若持有期限为 $k$ 年，则平均年化收益率为 $(1+R_t[k])^{1/k} - 1$。当收益率较小时，可用一阶泰勒展开近似为 $\frac{1}{k} \sum_{j=0}^{k-1} R_{t-j}$。

**2. 连续复合收益率 / 对数收益率 (Log Return)**
*   **定义**：简单毛收益率的自然对数。
    $r_t = \ln(1+R_t) = \ln(\frac{P_t}{P_{t-1}}) = \ln(P_t) - \ln(P_{t-1})$
*   **多期可加性（极其重要）**：连续复合多期收益率**等于它所包含的各单期对数收益率之和**。
    $r_t[k] = \ln(1+R_t[k]) = r_t + r_{t-1} + \dots + r_{t-k+1}$
    *（注：这一加法性质是金融建模中广泛使用对数收益率而不用简单收益率的根本原因）*

**3. 复利计息与现值/终值计算 (连续复合的金融意义)**
*   **离散复利**：假设年利率为 $r$，本金为 $C$。如果银行一年付息 $m$ 次，那么每次支付利息为 $r/m$。一年后存款的净值（终值）变成：
    $A = C \left(1 + \frac{r}{m}\right)^m$
*   **连续复利（极限情形）**：当付息次数 $m \to \infty$ 时（如按天甚至连续复利），根据极限定理 $\lim_{m \to \infty} (1+\frac{r}{m})^m = e^r$。
    *   一般地，**连续复合的资产终值** $A$ 公式为：
        $$A = C \cdot \exp(rn)$$
    *   反之，$n$ 年后价值为 $A$ 的资产的**现值** $C$（贴现公式）为：
        $$C = A \cdot \exp(-rn)$$

**4. 资产组合收益率**
*   资产组合的简单净收益率是各单项资产净收益率的加权平均：$R_{p,t} = \sum_{i=1}^N w_i R_{it}$。
*   *注意*：对数收益率的资产组合加权只是一个**近似等式**（$r_{p,t} \approx \sum w_i r_{it}$），严格意义上并不精确成立。

**5. 包含分红的收益率与超额收益率**
*   **含分红 ($D_t$) 的收益率**：$R_t = \frac{P_t + D_t - P_{t-1}}{P_{t-1}}$，连续复利则为 $r_t = \ln(P_t + D_t) - \ln(P_{t-1})$。
*   **超额收益率**：资产收益率与参考基准（如无风险利率 $R_{0t}$）之差：$Z_t = R_t - R_{0t}$。

### 二、 统计分布基础与随机变量的矩
为了深刻刻画收益率分布，必须掌握联合分布、边际分布的概念，以及随机变量各种矩（Moment）的严格定义：

**1. 联合分布、边际分布与条件分布**
*   **联合分布 (Joint Distribution)**：
    定义为 $F(x, y; \vartheta) = \Pr(X \le x, Y \le y)$。其中 $\vartheta$ 为分布参数。若概率密度函数 $f(x, y; \vartheta)$ 存在，则联合分布函数即为密度的双重积分。
*   **边际分布 (Marginal Distribution)**：
    描述多维随机变量中**某一个变量**单独的概率分布。例如，将联合分布中与 $X$ 不相关的赋值设为 $+\infty$，即得到 $X$ 的累积分布函数（CDF）：
    $$F_X(x; \vartheta) = \Pr(X \le x) = F(x, +\infty; \vartheta)$$
    CDF 经常用于计算统计检验中的 p-value。
*   **条件分布 (Conditional Distribution)**：
    给定某条件（如 $Y \le y$）下的分布。其概率密度定义为：
    $$f_{X|Y}(x | y; \vartheta) = \frac{f(x, y; \vartheta)}{f_Y(y; \vartheta)}$$
    在金融时间序列中，ARIMA、GARCH 等模型本质上都是在对条件期望 $E(X_t | X_{t-1}, \dots)$ 和条件方差 $Var(X_t | X_{t-1}, \dots)$ 进行建模。

**2. 随机变量矩的通用定义**
从统计学角度，一个随机变量 $X$ 的第 $l$ 阶矩定义为其减去均值 $\mu$ 的 $l$ 次方的期望（即中心矩，当 $\mu=0$ 时为原点矩）：
$$E[(X-\mu)^l] = \int_{-\infty}^{+\infty} (x-\mu)^l f(x) dx$$

**3. 随机变量的四阶矩物理含义**
对于均值为 $\mu$ 的随机变量 $X$：
*   **一阶矩（均值 / 期望）**：$E(X) = \mu$。度量分布的中心位置。
*   **二阶中心矩（方差，Variance）**：$Var(X) = E[(X-\mu)^2] = \sigma^2$。度量分布的离散程度（代表金融资产的波动率/风险）。
*   **三阶中心矩（偏度，Skewness）**：$E[(X-\mu)^3]$。
    *   度量分布的**非对称程度**。偏度大于 0 为右偏（长尾巴在右），小于 0 为左偏。
*   **四阶中心矩（峰度，Kurtosis）**：$E[(X-\mu)^4]$。
    *   度量分布的**尖峰程度或平坦程度**（尾部的厚度）。标准正态分布的峰度恒为 3。

**4. 期望的运算规则考点**
在推导 GARCH 的无条件方差等数学证明时，必须熟练运用以下基于白噪声扰动 $\varepsilon_t \sim iid(0, \sigma^2)$ 的期望法则（其中 $A$ 为常数，$u_t$ 与 $\varepsilon_t$ 独立）：
*   $E(\varepsilon_t) = 0$
*   $E(A\varepsilon_t) = A E(\varepsilon_t) = 0$
*   $Var(A\varepsilon_t) = A^2 Var(\varepsilon_t)$
*   和的期望可加：$E(A + \varepsilon_t) = A + E(\varepsilon_t) = A$
*   两项独立的项之和的方差：$Var(u_t + \varepsilon_t) = Var(u_t) + Var(\varepsilon_t)$
*   独立交叉项期望为 0：$E(u_t \varepsilon_t) = 0$，且 $Var(A + \varepsilon_t) = Var(\varepsilon_t)$
*   和的平方期望展开：$E[(u_t + \varepsilon_t)^2] = E(u_t^2) + E(\varepsilon_t^2) + 2E(u_t \varepsilon_t) = E(u_t^2) + E(\varepsilon_t^2)$

### 三、 资产收益率的常见理论分布假定
为了能够对 $r_t$ 进行参数估计和统计推断，学者们提出过多种收益率分布假定。对于 $N$ 个资产的对数收益率向量 $r_t = \{r_{1t}, \dots, r_{Nt}; t=1,\dots,T\}$，其最一般的联合分布函数模型为：
$$F(r_1, \dots, r_T; \theta | Y)$$
*(其中 $Y$ 是描述决定资产收益率环境的状态向量，$\theta$ 是决定分布函数的参数向量)*

在实际应用中，常用的单变量收益率理论分布假定有以下四种：

**1. 正态分布与对数正态分布**
*   **正态分布假定**：最早且最简单的假设是对数收益率服从正态分布，即 $r_t \sim N(\mu, \sigma^2)$。
*   **对简单收益率的推导**：由于 $r_t = \ln(1+R_t)$，若 $r_t \sim N(\mu, \sigma^2)$，则简单毛收益率 $1+R_t$ 服从**对数正态分布**。此时简单净收益率 $R_t$ 的均值 $m_1$ 和方差分别为：
    *   **均值**：$E(R_t) = m_1 = \exp(\mu + \frac{\sigma^2}{2}) - 1$
    *   **方差**：$Var(R_t) = \exp(2\mu + \sigma^2) \cdot [\exp(\sigma^2) - 1] = (1+m_1)^2 \cdot [\exp(\sigma^2) - 1]$
*   **缺陷**：虽然正态分布的性质好算（如任意线性组合仍为正态分布），但与真实的金融数据（普遍具有尖峰厚尾特征）严重不符。

**2. 稳定分布 (Stable Distribution / Lévy alpha-stable)**
*   **概念**：如果两个独立同分布的随机变量 $X_1$ 和 $X_2$ 的线性组合仍然是同样的分布（即 $aX_1 + bX_2 \sim cX + d$），则称该随机变量 $X$ 服从稳定分布（又称雷维阿尔法-稳定分布或分形分布）。
*   **四大决定参数**：稳定分布无法用简单的概率密度公式写出闭式解，而是由 4 个参数唯一定义：
    *   **尺度因子 $c$ ($c \neq 0$)**：描述分布的宽度（正态分布中的方差就是它）。
    *   **移位参数 $\mu \in (-\infty, +\infty)$**：描述分布的位置（正态分布中就是均值）。
    *   **幂指数 $\alpha \in (0, 2]$**：描述分布的渐进行为。$\alpha$ 越小，分布越平滑且尾部越厚；$\alpha$ 越大，概率密度函数中心越陡。**当 $\alpha=2$ 时，稳定分布退化为正态分布**。
    *   **偏度参数 $\beta \in [-1, 1]$**：衡量对称性。$\beta=0$ 时分布关于 $\mu$ 对称（雷维对称 $\alpha$-稳定分布）；当 $\beta \neq 0$ 时分布不对称（雷维偏 $\alpha$-稳定分布）。
        *(注意：当 $\beta=1$ 时，$\alpha$ 只能在 $(0, 1)$ 区间上取值)*
*   **三大经典特例及其概率密度函数（核心考点）**：
    1.  **Lévy 分布 (雷维分布)**：当 $\alpha = 1/2, \beta = 1$，$c$ 和 $\mu$ 任意取值时。由于 $\beta=1$，它是一个**极度非对称分布**。当 $\mu=0$ 时，随机变量仅在正半轴 $[0, +\infty)$ 上取值。
        *   **概率密度函数 (PDF)**：
            $$f(x; \mu, c) = \sqrt{\frac{c}{2\pi}} \frac{1}{(x-\mu)^{3/2}} \exp\left( - \frac{c}{2(x-\mu)} \right), \quad x > \mu$$
    2.  **Cauchy 分布 (柯西分布)**：当 $\alpha = 1, \beta = 0$ 时。它是一个**对称分布**，但尾部极厚，其数学期望和方差都不存在。
        *   **概率密度函数 (PDF)**：
            $$f(x; \mu, c) = \frac{1}{\pi} \left[ \frac{c}{(x-\mu)^2 + c^2} \right]$$
    3.  **正态分布 (Normal distribution)**：当 $\alpha = 2$ 时。此时偏度参数 $\beta$ 失去作用（对分布形状没有影响），分布完美对称且薄尾。
        *   **概率密度函数 (PDF)**：
            $$f(x; \mu, \sigma) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left( - \frac{(x-\mu)^2}{2\sigma^2} \right)$$

**3. 正态分布的尺度混合 (Scale Mixture of Normal Distributions)**
*   **物理意义**：现实中，金融市场可能在不同时期处于不同的“状态”（如平稳期和剧烈震荡期）。如果收益率分布由两个或多个方差不同的正态分布按一定概率混合而成，就能较好地模拟出“高峰厚尾”现象。
*   **数学表达**：例如一个两状态混合模型：
    $$r_t \sim (1 - X) \cdot N(\mu, \sigma_1^2) + X \cdot N(\mu, \sigma_2^2)$$
    *(其中 $X$ 是一个伯努利随机变量，$P(X=1) = p$ 代表发生罕见大波动（方差为 $\sigma_2^2$，且 $\sigma_2^2 \gg \sigma_1^2$）的概率)*

### 四、 真实金融时间序列的三大“程式化事实” (Stylized Facts)
通过对真实股票指数（如上证综指）的回报率进行实证分析（例如某样本偏度为0.75，峰度高达8.91，远超正态分布的3），人们发现真实的金融时间序列具有三大根本特征。**这三大特征正是导致传统经典线性回归模型彻底失效，从而逼迫计量经济学引入 ARCH/GARCH 族模型的根本原因：**

**1. 尖峰厚尾 (Leptokurtosis and Fat-tails)**
*   **现象**：真实的收益率分布曲线相比于标准正态分布，在均值附近表现出**过度集中（尖峰，Excess peakedness）**，而在两侧极端区域的概率远比正态分布大**（厚尾/重尾）**。
*   **含义**：金融市场出现极端暴涨或暴跌的概率，远大于正态分布所预测的概率。

**2. 波动丛集性 (Volatility Clustering / Pooling)**
*   **现象**：资产的波动率表现出强烈的**自相关性**。“大波动后面跟着大波动，小波动后面跟着小波动”。
*   **含义**：过程的方差不仅随时间变化，而且有时变化得很激烈（条件异方差），这正是 ARCH 建模的切入点。

**3. 正负冲击的非对称性 (杠杆效应)**
*   **现象**：好消息（正冲击）和坏消息（负冲击）对投资者和市场波动的影响并不对称。
*   **含义**：通常情况下，资产价格下跌（坏消息）带来的恐慌和随后的波动率放大程度，要显著高于等量价格上涨带来的波动率变化。

**⚠️ 总结对策：**
由于这三大特征的存在，必须摒弃传统的普通最小二乘法（OLS）。现代金融计量学给出的解决方案主要有两条路径：
1.  **动态条件方差建模**：使用 ARCH、GARCH 及其变体（如 EGARCH 解决非对称性）来精准拟合“波动丛集性”。
2.  **替换分布假定**：在 GARCH 模型的残差项上，放弃正态分布假定，改用能自带厚尾属性的 **t-分布**、**广义误差分布 (GED)** 或 **g&h 分布**等来拟合“尖峰厚尾”。
